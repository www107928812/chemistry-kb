# 中考化学实验探究

> 中考实验探究题答题思路与模板 | 补充自中考同步+一轮讲义

## 实验探究题答题模板

### 核心步骤
**提出问题 → 猜想与假设 → 设计实验 → 进行实验 → 观察现象 → 解释与结论 → 反思与评价 → 表达与交流**

### 猜想与假设
- 依据已有知识和经验
- 考虑反应的过量问题（恰好完全反应/某反应物过量）
- 考虑多种可能性

### 变量控制法
| 探究目的 | 控制变量方法 |
|----------|-------------|
| 燃烧条件 | 控制可燃物/O₂/温度三个变量，每次变一个 |
| 金属锈蚀条件 | 控制 O₂ 和水两个变量 |
| 催化剂效果 | 控制用量、温度、浓度等相同 |
| 反应速率影响 | 控制温度/浓度/接触面积/催化剂 |

### 实验评价角度
- 操作是否简便
- 现象是否明显
- 是否环保（尾气处理）
- 是否安全
- 是否经济

## 一、中考实验探究常见题型

### 题型一：物质组成的探究
- 可燃物燃烧产物分析（检验 H₂O → 含H；检验 CO₂ → 含C）
- 金属锈蚀条件探究（O₂ + H₂O → 控制变量法）

### 题型二：反应后溶液中溶质成分探究
- 中和反应后溶液成分 → 三种可能：恰好完全 / 酸过量 / 碱过量
- **思路**：猜想 → 设计实验验证 → 现象 → 结论

### 题型三：气体的检验与除杂
- 检验 H₂O → 无水 CuSO₄（白色→蓝色）
- 检验 CO₂ → 澄清石灰水变浑浊
- 除 CO₂ → NaOH 溶液
- 干燥气体 → 浓 H₂SO₄（酸性气体）/ 碱石灰（碱性气体）

### 题型四：控制变量法实验

**核心原则**：一次只改变一个因素，其他条件保持不变。

常考实验：
- 燃烧条件的探究（可燃物 / O₂ / 温度达着火点）
- 铁生锈条件的探究（O₂ / H₂O / 酸/盐环境）
- MnO₂ 对 H₂O₂ 分解速率的影响

## 二、实验方案设计模板

### 假设验证型
```
【提出问题】→ 【猜想与假设】→ 【设计实验】→ 【实验现象】→ 【实验结论】→ 【反思与评价】
```

### 条件探究型
- 设计对比实验
- 控制单一变量
- 观察现象 → 分析 → 结论

## 三、常见实验操作目的与答题规范

| 操作 | 目的 |
|------|------|
| 粉碎/研磨 | 增大接触面积，加快反应速率 |
| 搅拌 | 加快溶解/使受热均匀 |
| 过滤 | 分离不溶性固体与液体 |
| 蒸发时不断搅拌 | 防止液体局部过热而飞溅 |
| 制O₂结束时先撤导管再熄灭酒精灯 | 防止水倒流炸裂试管 |
| 铁丝燃烧瓶底放水/沙 | 防止高温熔融物溅落炸裂瓶底 |

## 四、实验评价与反思

### 常考角度
1. 实验方案是否环保（尾气处理）
2. 实验设计是否严谨（是否设置对照组）
3. 实验结果是否准确（误差分析）
4. 是否可以改进（简化操作/降低成本/减少污染）
