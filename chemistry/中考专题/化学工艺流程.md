# 专题突破三：化学工艺流程

> 来源：中考化学同步+一轮讲义

## 工艺流程题解题思路

### 读图方法
1. **找原料**（进）→ 原料是什么？需要除去什么杂质？
2. **找产品**（出）→ 目的是制取什么？
3. **看核心反应** → 哪一步是关键转化？
4. **看循环** → 哪些物质循环使用了？

### 常考操作及目的
| 操作 | 目的/要点 |
|------|----------|
| 粉碎/研磨 | 增大接触面积，加快反应速率 |
| 过滤 | 分离固体与液体 |
| 蒸发浓缩 | 加热蒸发溶剂 |
| 降温结晶 | 得到晶体（溶解度随温度变化大的物质） |
| 蒸发结晶 | 得到晶体（溶解度随温度变化小的物质，如NaCl） |
| 洗涤 | 除去晶体表面可溶性杂质 |
| 干燥 | 除去水分 |
| 高温煅烧 | 使碳酸盐分解或发生高温反应 |
| 通入过量气体 | 使反应充分进行 |

### 常考问答模式
- **"过量"的目的**：使反应充分/完全除去某物质
- **"适量"的原因**：避免引入新杂质
- **循环利用的物质**：看图中有回流箭头的物质
- **副产品的用途**：结合物质性质回答
- **产品纯度分析**：是否有未反应的原料残留

### 常见命题素材
- 粗盐提纯（溶解→过滤→蒸发→除杂）
- 金属回收（从废液/废料中提取金属）
- 制备物质（如 CaCO₃、Na₂CO₃ 的工业制备）
- 石灰石产业链
