# 中考化学计算

> 中考计算题专项训练 | 补充自中考同步+一轮讲义

## 一、化学式计算

### 基本公式
- 相对分子质量 = 各原子 Ar × 原子个数之和
- 元素质量比 = 各元素 (Ar × 原子个数) 之比
- 元素质量分数 = (该元素原子总质量 / 相对分子质量) × 100%
- 元素质量 = 物质质量 × 元素质量分数

### 混合物中元素计算
- 样品纯度（质量分数）= 纯物质质量 / 样品质量 × 100%
- 混合物中某元素质量分数 → 结合化学式推算

## 二、化学方程式计算（中考主考）

### 解题步骤
1. 写出配平的化学方程式
2. 标注已知量和未知量的相对分子质量
3. 列比例式
4. 求解
5. 写答案

### 常见计算类型
| 类型 | 解题关键 |
|------|---------|
| 直接计算 | 化学方程式中质量比例 |
| 含杂质计算 | 先求纯净物质量 = 混合物质量 × 纯度 |
| 过量判断 | 判断哪种反应物过量，以不足量为准 |
| 多步反应 | 找出中间产物的关系式 |
| 表格数据题 | 分析每次数据变化，找恰好完全反应的点 |
| 图像数据题 | 从图像拐点读取恰好反应的量 |

### 质量守恒定律在计算中的应用
- 反应前总质量 = 反应后总质量
- 差量法：固体质量差 / 溶液质量差 → 推算反应量
- 元素守恒：反应前后同种元素质量不变

### 典型题型
1. **直接计算**：已知一种物质质量求另一种
2. **含杂质计算**：先算纯净物质量 = 样品质量 × 纯度
3. **溶液中的计算**：溶质质量 = 溶液质量 × 质量分数

## 三、溶质质量分数计算

- ω = m(溶质) / m(溶液) × 100%
- m(溶液) = m(溶质) + m(溶剂)
- 稀释：m₁ω₁ = m₂ω₂（稀释前后溶质质量不变）

## 四、溶解度相关计算

- S / 100g = m(溶质) / m(溶剂)
- 饱和溶液中：ω = S / (S+100) × 100%

## 五、元素质量计算

- 化合物中某元素质量分数 = (该元素原子总质量 / 相对分子质量) × 100%
- 例：NH₄NO₃ 中氮元素质量分数 = (2×14)/(14×2+1×4+16×3)×100% = 35%
