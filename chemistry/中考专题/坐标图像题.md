# 专题突破四：坐标图像题

> 来源：中考化学同步+一轮讲义

## 图像题分类及读图方法

### 1. 金属与酸反应曲线
- 等质量金属+足量酸：化合价相同时，相对原子质量越小，产生 H₂ 越多
- 等量酸+足量金属：产生 H₂ 质量相等
- **斜率**表示反应速率（金属活动性强弱），**平台高度**表示氢气产量

### 2. 酸碱中和反应曲线
- pH 从酸性→中性→碱性（或反向）
- pH=7 时恰好完全反应
- 关注曲线突跃段

### 3. 溶解度曲线
- 大多数固体溶解度随温度升高而增大
- 少数（如 NaCl）基本不变
- 极少数（如 Ca(OH)₂）随温度升高而减小
- 交点表示温度相同时溶解度相等
- **结晶方法选择**：斜率大→降温结晶；斜率小→蒸发结晶

### 4. 反应中物质质量变化
| 反应类型 | 图像形状 |
|----------|---------|
| 金属+酸 | H₂ 质量从 0 增加到平台 |
| 向混合溶液中加试剂 | 可能存在分段反应（先发生优先反应） |
| 煅烧固体 | 固体质量逐渐减少至恒定 |

### 5. 常见易错
- 起点是否为零？（混合物反应）
- 终点是否为零？（过量问题）
- 斜率代表什么？（速率/效率）
- 平台代表什么？（反应完成/饱和）
