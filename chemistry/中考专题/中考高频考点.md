## 二、中考高频化学方程式（必背）

### 制取气体
- 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑
- 2H₂O₂ →MnO₂→ 2H₂O + O₂↑
- CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- Zn + H₂SO₄ → ZnSO₄ + H₂↑

### 碳和碳的氧化物
- C + O₂ →点燃→ CO₂
- 2CO + O₂ →点燃→ 2CO₂
- CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O
- Fe₂O₃ + 3CO →高温→ 2Fe + 3CO₂

### 金属
- Fe + CuSO₄ → FeSO₄ + Cu
- 3Fe + 2O₂ →点燃→ Fe₃O₄
- 2Al + 3H₂SO₄ → Al₂(SO₄)₃ + 3H₂↑

### 酸碱盐
- NaOH + HCl → NaCl + H₂O
- Ca(OH)₂ + Na₂CO₃ → CaCO₃↓ + 2NaOH
- Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑

## 三、必考知识点清单

### 空气成分
- N₂ 78%、O₂ 21%、稀有气体 0.94%、CO₂ 0.03%

### 电解水
- O₂:H₂ = 1:2（体积比），正氧负氢

### 金属活动性顺序（必背）
```
K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
```

### 溶解度曲线图读图
- 陡升型（如 KNO₃）→ 降温结晶
- 平缓型（如 NaCl）→ 蒸发结晶
- Ca(OH)₂ 溶解度随温度升高而**降低**（特殊）

### 常见沉淀颜色
- 白色：CaCO₃、BaCO₃、AgCl、BaSO₄、Mg(OH)₂、Al(OH)₃
- 蓝色：Cu(OH)₂
- 红褐色：Fe(OH)₃
- **不溶于酸**：AgCl、BaSO₄

## 四、中考推断题突破口

- 红色固体：Cu（紫红色）、Fe₂O₃（红棕色）、Fe(OH)₃（红褐色）
- 蓝色溶液/沉淀：含 Cu²⁺
- 浅绿色溶液：含 Fe²⁺
- 黄色溶液：含 Fe³⁺
- 能使石灰水变浑浊的气体：CO₂
- 最轻的气体：H₂
- 相对分子质量最小的氧化物：H₂O
- 常见的食品干燥剂：CaO
- 炉具清洁剂主要成分：NaOH

## 五、中考易错题归纳

1. **"CO₂ 能使紫色石蕊变红"** — 错！是 H₂CO₃ 使之变红
2. **铁与酸反应生成 Fe³⁺** — 错！铁与酸生成 Fe²⁺
3. **浓硫酸稀释** — 应酸入水！不能水入酸
4. **点燃 H₂/CO/CH₄ 前需验纯** — 否则爆炸
5. **催化剂"不参与反应"** — 错！参与反应，但质量和化学性质不变
6. **分离 NaCl 和 KNO₃** — KNO₃ 用降温结晶，NaCl 用蒸发结晶
