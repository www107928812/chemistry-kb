# 专题突破一：物质推断与转化

> 来源：中考化学同步+一轮讲义 | 中考必考推断题

## 推断题突破口（题眼）

### 1. 物质特征颜色
**固体**：
- 白色：BaSO₄、AgCl、CaCO₃、BaCO₃、Mg(OH)₂、Al(OH)₃、Na₂CO₃、NaOH、Ca(OH)₂
- 黑色：C、CuO、MnO₂、Fe₃O₄、铁粉
- 红色：Fe₂O₃（红棕色）、Cu（紫红色）
- 暗紫色：KMnO₄（溶液紫红色）

**溶液**：
- 蓝色（含 Cu²⁺）：CuCl₂、CuSO₄
- 浅绿色（含 Fe²⁺）：FeCl₂、FeSO₄
- 黄色（含 Fe³⁺）：FeCl₃、Fe₂(SO₄)₃

**沉淀**：
- 蓝色沉淀：Cu(OH)₂；红褐色沉淀：Fe(OH)₃
- 不溶于酸的白色沉淀：BaSO₄、AgCl
- 溶于酸且产生气体的白色沉淀：CaCO₃、BaCO₃
- 溶于酸无气体的白色沉淀：Mg(OH)₂、Al(OH)₃

### 2. 特征反应条件
| 条件 | 典型反应 |
|------|---------|
| 通电 | 电解水：2H₂O → 2H₂↑+O₂↑ |
| 催化剂 | KClO₃/MnO₂ 制 O₂；H₂O₂/MnO₂ 制 O₂ |
| 高温 | CaCO₃ 分解；C 还原 CuO；炼铁；C+CO₂→2CO |

### 3. 特征现象
- 使澄清石灰水变浑浊 → CO₂
- 火星四射 → Fe 在 O₂ 中燃烧
- 空气中燃烧生成 CO₂+H₂O → 有机物（CH₄、C₂H₅OH）
- 刺激性气味气体：HCl、NH₃、SO₂
- 可燃性气体：H₂、CO、CH₄

### 4. 物质之最
- 密度最小：H₂ | 硬度最大：金刚石 | 最简单有机物：CH₄
- 最常用溶剂：H₂O | 形成化合物最多：C
- 地壳最多元素：O；最多金属：Al | 年产量最高金属：Fe

### 5. 常见俗名
| 俗名 | 化学式 | 化学名 |
|------|--------|--------|
| 苏打/纯碱 | Na₂CO₃ | 碳酸钠 |
| 小苏打 | NaHCO₃ | 碳酸氢钠 |
| 烧碱/火碱/苛性钠 | NaOH | 氢氧化钠 |
| 熟石灰 | Ca(OH)₂ | 氢氧化钙 |
| 生石灰 | CaO | 氧化钙 |
| 干冰 | CO₂ | 固态二氧化碳 |
| 双氧水 | H₂O₂ | 过氧化氢 |
| 水煤气 | CO+H₂ | — |
| 大理石/石灰石 | CaCO₃ | 碳酸钙 |
| 铁锈/赤铁矿 | Fe₂O₃ | 氧化铁 |
| 胆矾/蓝矾 | CuSO₄·5H₂O | 硫酸铜晶体 |
| 食盐 | NaCl | 氯化钠 |

### 6. 常考物质转化关系
- **CO₂ ↔ CO**：CO₂+C→2CO；2CO+O₂→2CO₂
- **CaCO₃ ↔ CaO ↔ Ca(OH)₂**：CaCO₃→CaO+CO₂↑；CaO+H₂O→Ca(OH)₂；Ca(OH)₂+CO₂→CaCO₃↓+H₂O
- **Fe ↔ Fe²⁺ ↔ Fe³⁺** 转化
