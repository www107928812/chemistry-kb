# 高考化学反应原理综合题

> 高考必做非选择题 | 盖斯定律+平衡+图像分析

## 一、盖斯定律计算

### 方法
将已知热化学方程式像**数学等式**一样加减：
- 方程式翻倍 → ΔH 翻倍
- 方程式反写 → ΔH 反号
- 多个方程式相加 → ΔH 相加减

### 注意事项
- 标注物质状态（s, l, g, aq）
- ΔH 必须带正负号
- 化学计量数可用分数

## 二、平衡常数 K 的计算

### K 表达式
- 对于 aA + bB ⇌ cC + dD：K = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ
- K **只与温度有关**

### Kp（压强平衡常数）
- 用平衡时的分压代替浓度
- p(A) = (n(A)/n总) × p总

### 三段式法求 K
```
        A   +   B   ⇌   C   +   D
起始：   a       b       0       0
转化：   x       x       x       x
平衡：  a-x     b-x      x       x
```
代入 K 表达式求解

## 三、速率与平衡图像分析

### 常见图像类型
1. **v-t 图**（速率-时间）：条件改变瞬间速率跳变，根据 v正 vs v逆 判断平衡移动方向
2. **c-t 图**（浓度-时间）：水平段 = 平衡
3. **转化率-T 图**（转化率-温度）：放热反应先升后降（平衡移动 > 速率影响时）
4. **转化率-P 图**（转化率-压强）

### 图像分析方法
1. 看坐标轴含义
2. 看曲线走势（上升/下降）
3. 看多个变量（控制变量法分析）

## 四、平衡移动判断

### 勒夏特列原理
改变影响平衡的条件，平衡向**减弱**这种改变的方向移动

### 常见设问角度
- 升高温度对平衡的影响 → 向吸热方向
- 增大压强对平衡的影响 → 向气体分子数减少方向
- 通入惰性气体 → 恒容无影响，恒压向分子数增大方向
