# 高考化学化工流程题

> 高考必做非选择题 | 化工流程题答题模板

## 一、流程题分析思路

1. **原料预处理**：粉碎/酸浸/碱浸/焙烧 → 目的分析
2. **核心转化**：氧化还原/沉淀/置换 → 化学方程式书写
3. **分离提纯**：过滤/萃取/结晶 → 操作名称及目的
4. **产品获得**：结晶方式/干燥条件

## 二、常见操作及其答题术语

| 操作 | 答题术语 |
|------|---------|
| 粉碎/研磨 | 增大接触面积，加快反应速率，提高浸出率 |
| 酸浸 | 使金属/金属氧化物溶解，进入溶液 |
| 碱浸 | 使两性物质（Al、Al₂O₃等）溶解 |
| 焙烧 | 使物质在高温下反应/分解 |
| 调节pH | 促进某离子水解生成沉淀而除去 |
| 趁热过滤 | 防止某物质因温度降低而析出（或防止结晶） |
| 冷却结晶 | 降低温度使溶解度减小而析出晶体 |
| 洗涤（水洗） | 除去晶体表面的可溶性杂质 |
| 洗涤（醇洗） | 除去水分、快速干燥（利用酒精的挥发性） |
| 蒸发浓缩→冷却结晶→过滤→洗涤→干燥 | 标准流程 |
| 蒸发结晶→趁热过滤 | 获得溶解度受温度影响小的晶体 |

## 三、常考化学方程式

### 与酸/碱反应
- Al₂O₃ + 6H⁺ → 2Al³⁺ + 3H₂O
- Fe₂O₃ + 6H⁺ → 2Fe³⁺ + 3H₂O
- CuO + 2H⁺ → Cu²⁺ + H₂O
- SiO₂ + 2NaOH → Na₂SiO₃ + H₂O

### 调节 pH 除杂
- Fe³⁺ + 3H₂O ⇌ Fe(OH)₃↓ + 3H⁺
- Al³⁺ + 3H₂O ⇌ Al(OH)₃↓ + 3H⁺
- Cu²⁺ + H₂S → CuS↓ + 2H⁺

### 氧化还原
- 2Fe²⁺ + H₂O₂ + 2H⁺ → 2Fe³⁺ + 2H₂O
- 2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻

## 四、Ksp 相关计算

- 当某离子沉淀完全（浓度 < 10⁻⁵ mol/L）时所需的 pH 范围
- 判断能否通过调节 pH 实现离子分离
