# 高考化学实验综合

> 高考必做非选择题 | 综合实验题答题策略

## 一、实验装置与仪器

### 常见仪器连接顺序
**发生装置 → 净化装置（除杂）→ 干燥装置 → 主反应/收集 → 尾气处理**

### 常见装置作用
| 装置 | 作用 |
|------|------|
| 洗气瓶（长进短出） | 洗气/除杂/干燥 |
| 干燥管（球形/U形） | 固体干燥/检验气体 |
| 安全瓶 | 防倒吸 |
| 尾气吸收 | 防止有毒气体排放 |
| 量气装置 | 测定气体体积 |

## 二、气体制备与检验

### 常见气体综合制备

| 气体 | 反应原理 | 除杂 | 干燥 | 收集 |
|------|---------|------|------|------|
| O₂ | KMnO₄△/H₂O₂+MnO₂ | — | 浓H₂SO₄ | 排水/向上排空气 |
| CO₂ | CaCO₃+HCl | NaHCO₃(除HCl) | 浓H₂SO₄ | 向上排空气 |
| Cl₂ | MnO₂+浓HCl△ | 饱和食盐水(除HCl) | 浓H₂SO₄ | 向上排空气 |
| NH₃ | NH₄Cl+Ca(OH)₂△ | — | **碱石灰**(不能用CaCl₂) | **向下排空气** |

## 三、实验方案设计模板

### 检验/测定方案
1. 取样 → 2. 加试剂 → 3. 操作 → 4. 描述现象 → 5. 得出结论

### 注意规范表述
- "检验"：取样→加试剂→现象→结论
- "鉴别"：取样→加试剂（或其它操作）→分析不同现象→鉴别
- "除杂"：方法+不能引入新杂质

## 四、误差分析

常用公式：产率 = 实际产量/理论产量 × 100%

可能导致偏低的因素：
- 反应不完全
- 转移过程中洒落
- 副反应消耗原料
