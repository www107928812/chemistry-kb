**题目：** 在X、Y、Z三种金属中，如果把X和Y分别放入稀硫酸中，X溶解并产生气泡，Y不反应；如果把Y和Z分别放入硝酸银溶液中，在Y表面有固体物质析出，而Z没有变化。根据以上实验事实，判断这三种金属的活动性由强到弱的顺序为（ ）
A．X、Y、Z
B．Z、Y、X
C．Y、X、Z
D．X、Z、Y

**解析：** X能置换酸中氢→X>H；Y不能→Y<H。Y能置换Ag→Y>Ag；Z不能→Z<Ag。所以X>Y>Ag>Z。

**答案：** A

**题目：** 下列各组物质能验证Al、Cu、Ag三种金属活动性强弱的是（ ）
A．Al、Cu、Ag、稀盐酸
B．Cu、Al₂(SO₄)₃溶液、AgNO₃溶液
C．Al、Cu、AgNO₃溶液
D．Ag、Al₂(SO₄)₃溶液、Cu(NO₃)₂溶液

**解析：** B方案：Cu能置换AgNO₃中的Ag→Cu>Ag；Cu不能置换Al₂(SO₄)₃中的Al→Al>Cu。综合得Al>Cu>Ag。

**答案：** B

**题目：** 向盛有AgNO₃和Cu(NO₃)₂混合溶液的试管中加入一定量的铁粉，充分反应后过滤，滤液呈蓝色。下列判断正确的是（ ）
A．滤渣可能含有铜
B．滤液的质量可能比原溶液的质量大
C．滤液中可能含有Fe(NO₃)₂和Cu(NO₃)₂
D．向滤渣中加入稀盐酸，可能有气泡产生

**解析：** 滤液呈蓝色说明Cu²⁺未被完全置换。铁先置换Ag⁺后置换Cu²⁺。滤液可能含Fe(NO₃)₂和剩余的Cu(NO₃)₂以及可能未反应的AgNO₃。

**答案：** C

**题目：** 小明用三枚洁净无锈的铁钉，设计如图所示实验，探究铁生锈的条件。下列说法错误的是（ ）
A．乙试管中植物油的作用是隔绝空气
B．只有甲试管中的铁钉有明显锈迹
C．乙丙两支试管的实验现象说明铁生锈需要水

**答案：** C
