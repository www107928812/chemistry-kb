**题目：** 下列有关溶液的说法正确的是（ ）
A．溶液都是无色透明的
B．溶液中的溶质只能是一种
C．溶液是均一、稳定的混合物
D．均一、稳定的液体一定是溶液

**解析：** A错误——CuSO₄溶液是蓝色的；B错误——溶质可以多种；D错误——水是均一稳定的但不是溶液。

**答案：** C

**题目：** 甲、乙两种固体物质的溶解度曲线如图所示。下列说法正确的是（ ）
A．甲的溶解度大于乙的溶解度
B．t₁℃时，甲和乙的饱和溶液中溶质质量分数相等
C．将t₂℃时甲的饱和溶液降温至t₁℃，溶液质量不变
D．乙中混有少量甲时，可用降温结晶的方法提纯乙

**解析：** A需看温度；B正确——交点处溶解度相等→饱和溶液质量分数相等；C降温会析出晶体；D乙的溶解度随温度变化小，应用蒸发结晶。

**答案：** B

**题目：** 将50g 20%的氯化钠溶液加水稀释成10%的溶液，需要加水的质量为多少克？

**解析：** 稀释前后溶质质量不变：50g×20%=(50g+m水)×10%，m水=50g。

**答案：** 50g
