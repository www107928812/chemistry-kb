**题目：** 在化学反应A+B→C+D中，已知25gA与10gB恰好完全反应生成5gC，则生成D的质量为（ ）
A．20g
B．30g
C．35g
D．无法确定

**解析：** 根据质量守恒定律：m(A)+m(B)=m(C)+m(D)，25+10=5+m(D)，m(D)=30g。

**答案：** B

**题目：** 化学反应前后，一定不变的是（ ）
①原子种类 ②原子数目 ③分子种类 ④分子数目 ⑤元素种类 ⑥物质总质量
A．①②③⑤
B．①②⑤⑥
C．①②④⑤⑥
D．①②③④⑤⑥

**解析：** 反应前后：原子种类、数目、质量不变→元素种类、质量不变→物质总质量不变。分子种类一定变，分子数目可能变。

**答案：** B

**题目：** 下列化学方程式书写及反应类型均正确的是（ ）
A．2Fe+6HCl=2FeCl₃+3H₂↑（置换反应）
B．CaCO₃+HCl=CaCl₂+H₂O+CO₂↑（复分解反应）
C．2KMnO₄→△→K₂MnO₄+MnO₂+O₂↑（分解反应）
D．CO₂+Ca(OH)₂=CaCO₃↓+H₂O（复分解反应）

**解析：** A中Fe与HCl反应生成FeCl₂非FeCl₃；B未配平；C正确；D不属于四种基本反应类型。

**答案：** C

**题目：** 实验室用含杂质20%的石灰石（主要成分CaCO₃）25g与足量稀盐酸反应，可制得CO₂多少克？（杂质不与盐酸反应）

**解析：** CaCO₃质量=25g×(1-20%)=20g
CaCO₃+2HCl=CaCl₂+H₂O+CO₂↑
100 : 44
20g : x
100/20=44/x，x=8.8g

**答案：** 8.8g
