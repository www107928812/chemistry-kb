**题目：** 下列变化属于化学变化的是（ ）
A．海水晒盐
B．风力发电
C．树叶腐烂
D．铁水铸锅

**解析：** 化学变化有新物质生成。A海水晒盐是蒸发结晶（物理变化），B风力发电是动能转化（物理变化），C树叶腐烂有新物质生成（化学变化），D铁水铸锅是状态改变（物理变化）。

**答案：** C

B．铁在潮湿的空气里生成铁锈
C．水沸腾变成水蒸气

**题目：** 为防止燃气泄漏造成危险，可在食堂或家庭中安装报警器。下列燃气的性质中，能确定安装报警器位置的是（ ）
A．毒性
B．密度
C．气味
D．溶解性

**解析：** 报警器安装位置取决于燃气的密度——若密度比空气大，泄漏后下沉，报警器应装低位；若密度比空气小，泄漏后上升，应装高位。这是物理性质决定应用的典型例子。

**答案：** B

A．试管
B．烧杯

D．量筒

**解析：** 配制溶液和加热较多量液体需用烧杯（垫石棉网）。试管只能盛少量试剂；集气瓶用于收集气体；量筒只用于量取液体，不能作反应容器。

**答案：** B

**题目：** 某学生用量筒量取液体时，量筒放平稳后仰视液面读得数值为19mL，倾倒部分液体后，又俯视液面，读得数值为10mL。则该学生实际倾倒的液体体积是（ ）
A．9 mL
B．小于9 mL
C．大于9 mL
D．无法判断

**解析：** 仰视读数偏小 → 实际体积 > 19mL；俯视读数偏大 → 实际体积 < 10mL。实际倒出量 = 初始实际体积 − 剩余实际体积 > 19 − 10 = 9mL。

**答案：** C
