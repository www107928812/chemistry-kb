**题目：** 除去食盐水中含有的CaCl₂和Na₂SO₄杂质，应加入试剂的正确顺序是（ ）
A．先加BaCl₂溶液，再加Na₂CO₃溶液，最后加稀盐酸
B．先加Na₂CO₃溶液，再加BaCl₂溶液，最后加稀盐酸
C．先加稀盐酸，再加BaCl₂溶液，最后加Na₂CO₃溶液
D．先加BaCl₂溶液，再加稀盐酸，最后加Na₂CO₃溶液

**答案：** A

## 考点五：离子共存

### TK-C9-U10-008
| 字段 | 内容 |
|------|------|
| 章节 | 九年级下册·第十单元 常见的酸、碱、盐 |
| 考点 | 离子共存（限定 pH+无色） |
| 题型 | 选择题 |
| 难度 | ★★☆ |
| 来源 | Claude 生成 |
| 对应知识点 | 第十单元-常见的酸碱盐.md |

**题目：** 下列各组离子在 pH=2 的无色溶液中能大量共存的是（ ）
A．Na⁺、CO₃²⁻、Cl⁻、K⁺
B．Cu²⁺、SO₄²⁻、Na⁺、NO₃⁻
C．Ba²⁺、Cl⁻、Na⁺、NO₃⁻
D．Ca²⁺、Na⁺、SO₄²⁻、NO₃⁻

**解析：** A：CO₃²⁻ 与 H⁺ 反应（2H⁺+CO₃²⁻→H₂O+CO₂↑）；B：Cu²⁺ 溶液为蓝色，不符"无色"；D：Ca²⁺+SO₄²⁻→CaSO₄（微溶，视作沉淀）。三大筛选顺序：有色排除→沉淀排除→酸碱反应排除。

**答案：** C

## 考点六：离子/酸碱盐推断

### TK-C9-U10-009
| 字段 | 内容 |
|------|------|
| 章节 | 九年级下册·第十单元 常见的酸、碱、盐 |
| 考点 | 离子推断（实验操作+排除法推理） |
| 题型 | 填空题 |
| 难度 | ★★★ |
| 来源 | Claude 生成 |
| 对应知识点 | 第十单元-常见的酸碱盐.md |

**题目：** 某无色溶液中可能含有 Na⁺、Ba²⁺、Cu²⁺、CO₃²⁻、SO₄²⁻、Cl⁻ 中的几种，进行如下实验：
① 取少量溶液，滴加酚酞溶液，溶液变红
② 另取少量溶液，加入足量稀盐酸，有气泡产生
③ 向②反应后的溶液中加入 BaCl₂ 溶液，无明显现象

请回答：
(1) 原溶液中一定存在的离子有\_\_\_\_\_，一定不存在的离子有\_\_\_\_\_
(2) 原溶液中可能存在的离子有\_\_\_\_\_
(3) 写出步骤②中发生反应的化学方程式\_\_\_\_\_

**解析：** ①酚酞变红→溶液呈碱性→CO₃²⁻存在；含CO₃²⁻则Ba²⁺、Cu²⁺不能共存（沉淀），且无色→无Cu²⁺。②加盐酸有气泡→确认CO₃²⁻（CO₃²⁻+2H⁺→H₂O+CO₂↑）。阳离子只剩Na⁺→Na⁺一定存在。③加BaCl₂无沉淀→无SO₄²⁻。步骤②③均引入Cl⁻→无法判断原溶液Cl⁻。

**答案：** (1)一定存在：Na⁺、CO₃²⁻；一定不存在：Ba²⁺、Cu²⁺、SO₄²⁻ (2)可能存在：Cl⁻ (3)Na₂CO₃+2HCl=2NaCl+H₂O+CO₂↑
