# 第三章 水溶液中的离子反应与平衡

> 人教版（2019）选择性必修1 | 高考原理综合题难点

## 第一节 电离平衡

### 弱电解质的电离
- 电离平衡常数 Ka（弱酸）/ Kb（弱碱）—— 只与温度有关
- 电离度 α = 已电离分子数 / 原有分子总数
- **越稀越电离，越热越电离**（吸热）

### 影响电离平衡的因素
- 稀释：促进电离，但离子浓度不一定增大
- 升温：促进电离（吸热）
- 同离子效应：抑制电离

## 第二节 水的电离和溶液的 pH

### 水的电离
- H₂O ⇌ H⁺ + OH⁻
- Kw = [H⁺][OH⁻] = 1×10⁻¹⁴（25℃）
- **温度升高，Kw 增大**（水的电离吸热）

### pH
- pH = -lg[H⁺]
- pH + pOH = 14（25℃）

### 溶液pH的计算
- **强酸**：pH = -lg[H⁺]
- **强碱**：先算[OH⁻] → [H⁺] = Kw/[OH⁻] → pH
- **混合**：先判断过量 → 算过量离子浓度 → 算pH
- **稀释**：强酸稀释10ⁿ倍→pH增大n；强碱稀释10ⁿ倍→pH减小n（趋近7但不越过7）

### 酸碱中和滴定
| 滴定类型 | 指示剂 | 变色范围 |
|----------|--------|---------|
| 强酸滴强碱 | 甲基橙或酚酞 | — |
| 强酸滴弱碱 | 甲基橙 | 3.1~4.4 |
| 强碱滴弱酸 | 酚酞 | 8.0~10.0 |

### 滴定误差分析（用 c = n/V 判断）
- 滴定管未用标准液润洗 → 结果偏高
- 锥形瓶用待测液润洗 → 结果偏高
- 尖嘴有气泡 → 结果偏高

## 第三节 盐类的水解

### 水解规律
**口诀**：**谁弱谁水解，谁强显谁性，越弱越水解，越热越水解**

| 盐 | 酸碱性 | 原因 |
|----|--------|------|
| NH₄Cl | 酸性 | NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺ |
| Na₂CO₃ | 碱性 | CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻ |
| CH₃COONa | 碱性 | CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻ |
| FeCl₃ | 酸性 | Fe³⁺ + 3H₂O ⇌ Fe(OH)₃ + 3H⁺ |
| NaCl | 中性 | 不水解 |

### 双水解（完全水解）
- Al³⁺ + 3HCO₃⁻ → Al(OH)₃↓ + 3CO₂↑（**泡沫灭火器原理**）
- Fe³⁺ + 3HCO₃⁻ → Fe(OH)₃↓ + 3CO₂↑

### 溶液中离子浓度大小比较（三大守恒）

**电荷守恒**：溶液中阳离子所带正电荷总数 = 阴离子所带负电荷总数

**物料守恒**：某元素的原子总数 = 含该元素的各种粒子中该原子的总数

**质子守恒**：溶液中得质子后的产物 = 失质子后的产物

以 Na₂CO₃ 为例：[Na⁺] > [CO₃²⁻] > [OH⁻] > [HCO₃⁻] > [H⁺]

## 第四节 沉淀溶解平衡

### 溶度积 Ksp
- MmAn(s) ⇌ mMⁿ⁺ + nAm⁻，Ksp = [Mⁿ⁺]ᵐ[Am⁻]ⁿ
- Ksp **只与温度有关**
- Qc > Ksp → 生成沉淀；Qc = Ksp → 饱和；Qc < Ksp → 不饱和

### 沉淀转化
- 一般：Ksp 大 → Ksp 小（越难溶的越先生成）
- 特例：Ksp 接近时也能实现逆向转化
- 应用：水垢处理、氟化物防龋齿
