# 第二章 化学反应速率与化学平衡

> 人教版（2019）选择性必修1 | 高考原理综合题核心

## 第一节 化学反应速率

### 速率表达式
- v = Δc / Δt，单位 mol/(L·s) 或 mol/(L·min)
- 对于 aA + bB → cC + dD：v(A):v(B):v(C):v(D) = a:b:c:d

### 影响因素
| 因素 | 对速率影响 | 原因 |
|------|-----------|------|
| 浓度 ↑ | 加快 | 单位体积内活化分子数↑ |
| 压强 ↑（气体） | 加快 | 相当于浓度↑ |
| 温度 ↑ | **显著加快** | 活化分子百分数↑ |
| 催化剂 | 加快 | 降低活化能（不改变ΔH） |
| 固体表面积 ↑ | 加快 | 接触面积↑，碰撞频率↑ |

### 速率常数（k）

**定义**：速率方程 v = k · [A]ᵐ · [B]ⁿ 中的比例系数，物理意义为各反应物为单位浓度时的反应速率
- k 与**浓度无关**，受**温度、催化剂、反应本性**影响
- k 的**单位随反应级数变化**：0级→mol·L⁻¹·s⁻¹ / 1级→s⁻¹ / 2级→L·mol⁻¹·s⁻¹

#### 基元反应 vs 复杂反应

| | 基元反应 | 复杂反应（非基元） |
|---|---------|-----------------|
| 步骤 | 一步完成 | 多步基元步骤组成 |
| 速率方程 | 直接按计量数写（**质量作用定律**） | **必须由实验测定** |
| 级数 | = 计量数之和（仅 1/2/3） | 可为整数/分数/零/负数 |
| 反应分子数 | 有意义 | 无意义 |
| k 的含义 | 单一步骤的真实 k | 多步复合的**表观 k** |
| Ea 含义 | 真实活化能 | **表观活化能** |
| 速控步 | — | 最慢步决定总速率 |

> **高中考点**：题目没说"基元反应"，就不能照抄计量数写速率方程。必须通过实验数据（控制变量法）求级数和 k。

**判断基元反应的方法**：
1. 测速率方程 → 级数 ≠ 计量数和 → 一定不是（级数为分数/零/负数直接排除）
2. 级数 = 计量数和 → **不一定**是（如 H₂+I₂→2HI，v=k[H₂][I₂] 恰好吻合，但实际分两步）
3. 计量数和 > 3 → 一定不是（四分子及以上碰撞概率≈零）
4. 题目会明说；高考不可能让学生自己判断是否是基元

**k 与 K 的关系**（基元可逆反应）：K = k正 / k逆

#### 阿伦尼乌斯公式（定量描述 k~T 关系）

$$ k = A \cdot e^{-E_a/RT} $$
| 符号 | 含义 |
|------|------|
| A | 指前因子（频率因子） |
| Ea | 活化能 (J/mol) |
| R | 8.314 J/(mol·K) |
| T | 热力学温度 (K) |

**两温度间比较**：
$$ \ln\frac{k_2}{k_1} = \frac{E_a}{R}\left(\frac{1}{T_1} - \frac{1}{T_2}\right) $$

**结论**：T↑ → k↑；催化剂降低 Ea → k↑

> **范托夫经验规律**：温度每升高 10K，溶液反应速率约增至 2~4 倍

### 活化能
- 活化分子具有的最低能量与分子平均能量之差
- 催化剂降低活化能，增大活化分子百分数，加快速率

## 第二节 化学平衡

### 化学平衡状态
**五大特征**：逆、等、定、动、变

### 化学平衡常数 K
- aA + bB ⇌ cC + dD，K = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ
- K **只与温度有关**
- K 越大，反应进行越完全
- Qc < K → 正向；Qc = K → 平衡；Qc > K → 逆向

### 影响化学平衡的因素

| 条件改变 | 平衡移动方向 | 对K影响 |
|----------|-------------|--------|
| 增加反应物浓度 | 正向 | 不变 |
| 升高温度（放热反应） | 逆向 | **减小** |
| 升高温度（吸热反应） | 正向 | **增大** |
| 增大压强（气体分子数↓） | 正向 | 不变 |
| 加催化剂 | **不移动** | 不变 |

### 勒夏特列原理
平衡向**减弱**外界条件改变的方向移动——但不能**抵消**改变

## 第三节 化学反应的方向

### 判据
- **ΔG = ΔH - T·ΔS**
- ΔG < 0 → 反应自发进行
- **焓减（ΔH<0）、熵增（ΔS>0）均有利于反应自发**

## 第四节 化学反应的调控

### 工业合成氨条件选择
N₂ + 3H₂ ⇌ 2NH₃（放热反应、气体分子数减少）
- **温度**：400-500℃（低温有利于平衡但速率太慢，故高温牺牲平衡争速率）
- **压强**：10-30MPa（高压有利于平衡和速率，但设备成本高）
- **催化剂**：铁触媒（加快速率但不影响平衡）
