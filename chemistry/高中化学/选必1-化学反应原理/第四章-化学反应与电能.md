# 第四章 化学反应与电能

> 人教版（2019）选择性必修1 | 电化学

## 第一节 原电池

### 工作原理
- 化学能 → 电能
- 较活泼金属 → **负极**（失电子，氧化反应）
- 较不活泼金属/石墨 → **正极**（得电子，还原反应）
- 电子：负极 →导线→ 正极
- 电流：正极 →导线→ 负极
- 电解质溶液中：阳离子 → 正极，阴离子 → 负极

### 常见化学电源
| 电池 | 负极 | 正极 | 电解质 |
|------|------|------|--------|
| 锌锰干电池 | Zn | MnO₂/C | NH₄Cl糊 |
| 铅蓄电池（可充电） | Pb | PbO₂ | H₂SO₄ |
| 氢氧燃料电池 | H₂ | O₂ | KOH/H₂SO₄ |

### 燃料电池
- 燃料在负极反应，O₂ 在正极反应
- 酸性介质：O₂ + 4H⁺ + 4e⁻ → 2H₂O
- 碱性介质：O₂ + 2H₂O + 4e⁻ → 4OH⁻

## 第二节 电解池

### 工作原理
- 电能 → 化学能
- **阳极**（接电源正极）：氧化反应
- **阴极**（接电源负极）：还原反应

### 放电顺序

**阳极（失电子）**：活泼金属阳极 > S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根 > F⁻

**阴极（得电子）**：Ag⁺ > Fe³⁺ > Cu²⁺ > H⁺(酸) > Pb²⁺ > Fe²⁺ > Zn²⁺ > H⁺(水) > Al³⁺ > Mg²⁺ > Na⁺

### 电解的应用
- 电解饱和食盐水（氯碱工业）：2NaCl + 2H₂O →电解→ 2NaOH + H₂↑ + Cl₂↑
- 电镀：待镀金属做阴极，镀层金属做阳极
- 精炼铜：粗铜做阳极，精铜做阴极
- 电解熔融 NaCl/ Al₂O₃ 制取活泼金属

## 第三节 金属的腐蚀与防护

### 电化学腐蚀（更普遍）

**析氢腐蚀（酸性环境）**：
- 负极：Fe - 2e⁻ → Fe²⁺
- 正极：2H⁺ + 2e⁻ → H₂↑

**吸氧腐蚀（中性/弱酸性/碱性，更常见）**：
- 负极：Fe - 2e⁻ → Fe²⁺
- 正极：O₂ + 4e⁻ + 2H₂O → 4OH⁻
- 最终生成 Fe₂O₃·xH₂O（铁锈）

### 金属防护方法
1. 涂保护层（油漆、电镀、烤蓝）
2. **牺牲阳极的阴极保护法**（如船体连接 Zn 块）
3. **外加电流的阴极保护法**（外加直流电源，被保护金属接负极）
