# 第五章 化工生产中的重要非金属元素

> 人教版（2019）必修第二册 | 硫、氮、硅及其化合物

## 第一节 硫及其化合物

### 硫 S
- 黄色固体，不溶于水，易溶于 CS₂
- S + O₂ →点燃→ SO₂（空气中淡蓝色火焰，纯氧中蓝紫色）

### SO₂
- 无色，有刺激性气味，有毒
- **漂白性**：与品红结合生成不稳定无色物质（**可逆**，区别于HClO的不可逆氧化漂白）
- **SO₂ 使品红褪色 → 加热后红色恢复**（鉴别SO₂和Cl₂！）
- 酸性氧化物：SO₂ + H₂O ⇌ H₂SO₃
- 较强还原性（+4→+6）：能被 O₂、Cl₂、KMnO₄ 等氧化
- 弱氧化性：SO₂ + 2H₂S → 3S↓ + 2H₂O

### SO₃
- SO₃ + H₂O → H₂SO₄（工业制硫酸）

### 浓 H₂SO₄ 三大特性

| 性质 | 现象 | 举例 |
|------|------|------|
| **吸水性** | 吸收水分 | 作干燥剂（不能干燥 NH₃、H₂S 等碱性/还原性气体） |
| **脱水性** | 使有机物碳化变黑 | 蔗糖→黑色面包实验（化学变化） |
| **强氧化性** | 与Cu/C反应 | Cu+2H₂SO₄(浓)→△→CuSO₄+SO₂↑+2H₂O |

> **吸水性 vs 脱水性**：吸水是吸收现成的水分子（物理变化为主）；脱水是从有机物中"夺取"H和O原子生成水（化学变化）。

### SO₄²⁻ 检验
先加**稀盐酸**，再加 BaCl₂ 溶液 → 白色沉淀（BaSO₄），不溶于盐酸

> 不能用稀硝酸或先加BaCl₂再加盐酸（可能有Ag⁺干扰）

## 第二节 氮及其化合物

### N₂
- 无色无味，占空气体积 ≈78%
- 结构稳定（N≡N 键能大，941 kJ/mol）

### NH₃
- 无色，有刺激性气味，密度小于空气
- **唯一使湿润红色石蕊试纸变蓝的气体**
- 极易溶于水（1体积水溶解700体积NH₃）→ 喷泉实验
- NH₃ + H₂O ⇌ NH₃·H₂O ⇌ NH₄⁺ + OH⁻（溶液显弱碱性）
- 实验室制取：2NH₄Cl + Ca(OH)₂ →△→ CaCl₂ + 2NH₃↑ + 2H₂O
- 与 HCl：NH₃ + HCl → NH₄Cl（产生**白烟**——相互检验）

### 铵盐
- 受热易分解：NH₄HCO₃ →△→ NH₃↑ + H₂O↑ + CO₂↑
- 与碱加热：NH₄⁺ + OH⁻ →△→ NH₃↑ + H₂O（检验NH₄⁺）
- NH₄NO₃ 是常用氮肥和炸药原料

### 氮的氧化物
| 化学式 | 名称 | N价态 | 特征 |
|--------|------|-------|------|
| N₂O | 一氧化二氮 | +1 | 笑气 |
| NO | 一氧化氮 | +2 | 无色，遇O₂→红棕色NO₂ |
| NO₂ | 二氧化氮 | +4 | 红棕色，刺激性气味 |
| N₂O₅ | 五氧化二氮 | +5 | HNO₃酸酐 |

- **NO 与 NO₂ 转化**：2NO + O₂ → 2NO₂；3NO₂ + H₂O → 2HNO₃ + NO

### HNO₃
- 浓硝酸有强氧化性
- 与金属反应**一般不产生 H₂**（产生 NO/NO₂）
- Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O
- 3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O
- 浓硝酸使 Fe/Al **钝化**（常温）

## 第三节 无机非金属材料

### 硅 Si
- 半导体材料（芯片、太阳能电池）
- 自然界以 SiO₂ 和硅酸盐形式存在

### SiO₂
- 原子晶体，硬度大，熔点高
- 酸性氧化物 + 特殊性：能与 **HF** 反应
- SiO₂ + 4HF → SiF₄↑ + 2H₂O（氢氟酸用于蚀刻玻璃）
- 与NaOH：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O

### 硅酸盐
- Na₂SiO₃：水溶液叫水玻璃（可作防火剂）
- 传统硅酸盐材料：陶瓷、玻璃、水泥

### S₈（0价）与 +4价硫（SO₂/Na₂SO₃）还原性对比

> 学生常见误区：价态越低还原性越强。此条专门破除。

**结构原因**：
- **S₈**：皇冠形八元环，S–S 键能较高，断开需要能量，常温下不易表现还原性（更易表现氧化性，如与 Fe、Cu 反应）
- **SO₃²⁻/SO₂**：中心硫原子有孤对电子，+6价硫酸根为正四面体稳定结构，+4价硫极易被氧化

**反应事实对比**：
| 物质 | 与O₂（空气） | 与卤素（Br₂水） |
|------|--------------|------------------|
| S（0价） | 需点燃才生成SO₂ | 不使溴水褪色（或反应极慢） |
| SO₂/Na₂SO₃（+4价） | 常温Na₂SO₃溶液变质生成SO₄²⁻ | 使溴水迅速褪色：SO₂ + Br₂ + 2H₂O → H₂SO₄ + 2HBr |

> **结论**：0价硫的还原性 < +4价硫的还原性。不能简单用"价态越低还原性越强"推导，须结合物质存在形态和结构稳定性。

### S²⁻ 氧化产物规律

H₂S 与一般氧化剂（Fe³⁺、I₂、O₂常温）反应仅生成 S 单质，不生成 SO₂。原因：S²⁻ → S → SO₂ 为分步氧化，常见氧化剂的氧化性仅大过 S 的氧化性但**大不过** SO₂ 的氧化性。

浓 HNO₃、足量 Cl₂（加热）等强氧化剂可进一步氧化到 SO₂ 甚至 H₂SO₄。
