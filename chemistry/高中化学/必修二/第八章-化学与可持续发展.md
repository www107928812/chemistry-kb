# 第八章 化学与可持续发展

> 人教版（2019）必修第二册

## 第一节 自然资源的开发利用

### 金属冶炼
| 方法 | 适用金属 | 举例 |
|------|---------|------|
| 电解法 | K~Al（活泼金属） | 2Al₂O₃ →电解→ 4Al + 3O₂↑ |
| 热还原法 | Zn~Cu | Fe₂O₃+3CO→2Fe+3CO₂ |
| 热分解法 | Hg、Ag | 2HgO→△→2Hg+O₂↑ |
| 物理法 | Pt、Au | 淘金 |

### 海水资源的开发利用
- 海水淡化：蒸馏法、电渗析法、离子交换法
- 海水制盐 → 氯碱工业
- 海水提溴、海水提镁

### 煤、石油的综合利用
- 煤：干馏（化学变化）、气化（C+H₂O→CO+H₂）、液化
- 石油：分馏（物理变化）、裂化/裂解（化学变化）

## 第二节 化学品的合理使用

### 化肥的安全使用
- 铵态氮肥不与碱性物质混用（NH₄⁺与OH⁻反应逸出NH₃）
- 合理施用（过量→水体富营养化）

### 农药
- 高效低毒低残留

### 常见食品添加剂
- 防腐剂、抗氧化剂、着色剂、调味剂等
- 在规定范围内使用是安全的

## 第三节 环境保护与绿色化学

### 主要环境问题
| 问题 | 主要污染物 |
|------|-----------|
| 酸雨 | SO₂、NOₓ（pH<5.6） |
| 温室效应 | CO₂ |
| 臭氧层破坏 | 氟氯代烷（氟利昂） |
| 光化学烟雾 | 氮氧化物+碳氢化合物 |
| 水体富营养化 | 含N/P废水 |
| PM2.5 | 可吸入颗粒物 |

### 绿色化学
- 核心思想：从源头上减少或消除污染
- 原子经济性：反应物中的原子全部转化为产物
- 理想原子利用率 = 100%
