## 第一节 化学反应与能量变化

### 放热反应 vs 吸热反应

| | 放热反应（ΔH<0） | 吸热反应（ΔH>0） |
|----|----|----|
| 举例 | 燃烧、中和、金属+酸、CaO+H₂O | Ba(OH)₂·8H₂O+NH₄Cl、C+CO₂、大多数分解反应 |
| 能量 | 化学能 → 热能 | 热能 → 化学能 |

- 构成条件：两个电极 → 电解质溶液 → 闭合回路 → 自发氧化还原反应

## 第二节 化学反应的速率与限度

### 化学反应速率
- v = Δc / Δt（单位：mol/(L·s) 或 mol/(L·min)）
- **影响因素**：
  - 温度↑ → 速率↑
  - 浓度↑ → 速率↑
  - 压强↑（有气体）→ 速率↑
  - 催化剂 → 改变速率
  - 固体表面积↑ → 速率↑

### 化学平衡
- 可逆反应中，v正 = v逆 时的状态
- 平衡特征：各组分浓度不再变化（但反应没有停止）
- **勒夏特列原理**：改变条件时，平衡向减弱这种改变的方向移动
