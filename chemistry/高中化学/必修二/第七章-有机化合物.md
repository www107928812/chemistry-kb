# 第七章 有机化合物

> 人教版（2019）必修第二册 | 高中有机化学入门

## 第一节 认识有机化合物

### 有机物的特征
- 都含碳元素（CO、CO₂、碳酸盐等属无机物）
- 多数难溶于水、易溶于有机溶剂、易燃

### 同系物与同分异构体
- **同系物**：结构相似，分子组成相差一个或若干个 CH₂
- **同分异构体**：分子式相同，结构不同（如正丁烷与异丁烷）

### 甲烷 CH₄
- 最简单的有机物，天然气/沼气主要成分
- 正四面体结构，键角 109°28'
- **取代反应**：CH₄ + Cl₂ →光→ CH₃Cl + HCl（可继续取代）
- **燃烧**：CH₄ + 2O₂ →点燃→ CO₂ + 2H₂O（淡蓝色火焰）
- 不能使酸性KMnO₄和溴水褪色

## 第二节 乙烯与有机高分子材料

### 乙烯 C₂H₄
- 石油化工的重要原料
- 平面结构，6原子共面，键角约 120°
- **加成反应**：CH₂=CH₂ + Br₂ → CH₂Br-CH₂Br（溴水褪色——检验 C=C）
- 加聚：nCH₂=CH₂ → [-CH₂-CH₂-]n（聚乙烯）
- 能使酸性 KMnO₄ 褪色（被氧化）

### 有机高分子材料
- **塑料**：聚乙烯(PE)、聚氯乙烯(PVC)、聚丙烯(PP)
- **合成纤维**：涤纶、尼龙
- **合成橡胶**：丁苯橡胶

## 第三节 乙醇与乙酸

### 乙醇 C₂H₅OH
- 俗称酒精，与水任意比例互溶
- 与Na反应：2C₂H₅OH + 2Na → 2C₂H₅ONa + H₂↑
- 催化氧化：2C₂H₅OH + O₂ →Cu/△→ 2CH₃CHO + 2H₂O
- 被强氧化剂氧化：可使酸性KMnO₄/酸性K₂Cr₂O₇褪色（橙色→绿色，查酒驾原理）

### 乙酸 CH₃COOH
- 食醋主要成分（3%~5%），弱酸
- 酯化反应：CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O
  - **酸脱羟基，醇脱氢**（同位素示踪法验证）

## 第四节 基本营养物质

### 糖类
| 分类 | 代表物 | 分子式 | 特征 |
|------|--------|--------|------|
| 单糖 | 葡萄糖、果糖 | C₆H₁₂O₆ | 葡萄糖含醛基（能银镜反应） |
| 二糖 | 蔗糖、麦芽糖 | C₁₂H₂₂O₁₁ | 互为同分异构体 |
| 多糖 | 淀粉、纤维素 | (C₆H₁₀O₅)n | n值不同，不是同分异构体 |

- 淀粉**遇碘变蓝**（检验淀粉）
- 水解最终产物均为葡萄糖

### 蛋白质
- 由氨基酸缩合形成
- **盐析**：加 NaCl/(NH₄)₂SO₄ → 析出（可逆，物理变化）
- **变性**：加热/重金属/酸/碱/酒精 → 永久失活（不可逆）
- 遇浓硝酸变黄（含苯环的蛋白质）

### 油脂
- 高级脂肪酸甘油酯
- 碱性水解（皂化）：油脂 + NaOH → 高级脂肪酸钠（肥皂）+ 甘油
