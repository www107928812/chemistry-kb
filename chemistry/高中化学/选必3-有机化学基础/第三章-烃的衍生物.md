# 第三章 烃的衍生物

> 人教版（2019）选择性必修3 | 高考有机核心

## 第一节 卤代烃

### 代表物：溴乙烷 C₂H₅Br

**水解反应（取代）**：
C₂H₅Br + NaOH →H₂O/△→ C₂H₅OH + NaBr

**消去反应**：
C₂H₅Br + NaOH →醇/△→ CH₂=CH₂↑ + NaBr + H₂O
**条件**：邻位碳上必须有氢（β-H）

### 卤代烃中卤原子的检验
NaOH水溶液加热水解 → 加稀硝酸中和 → 加AgNO₃溶液 → 观察沉淀颜色

---

## 第二节 醇 酚

### 醇 — 代表物：乙醇 C₂H₅OH

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与Na | 2C₂H₅OH + 2Na → 2C₂H₅ONa + H₂↑ | — |
| 催化氧化 → 乙醛 | 2C₂H₅OH + O₂ → 2CH₃CHO + 2H₂O | Cu/Ag, △ |
| 消去 → 乙烯 | C₂H₅OH → CH₂=CH₂↑ + H₂O | 浓H₂SO₄, 170℃ |
| 酯化 | C₂H₅OH + CH₃COOH ⇌ CH₃COOC₂H₅ + H₂O | 浓H₂SO₄, △ |

**醇的催化氧化规律**：
- 伯醇（—CH₂OH）→ 醛 → 羧酸
- 仲醇（—CHOH—）→ 酮（不再继续氧化）
- 叔醇（—COH，连OH的碳上没有H）→ 不能催化氧化

### 酚 — 代表物：苯酚 C₆H₅OH

| 性质 | 说明 |
|------|------|
| 弱酸性（石炭酸） | C₆H₅OH + NaOH → C₆H₅ONa + H₂O（苯酚钠） |
| 与Na | 2C₆H₅OH + 2Na → 2C₆H₅ONa + H₂↑ |
| 与溴水（取代） | 生成 2,4,6-三溴苯酚**白色沉淀** |
| 与FeCl₃ | **紫色**（显色反应，检验酚羟基） |
| 苯酚钠 + CO₂ + H₂O → 苯酚↓ + NaHCO₃ | 证明酸性：H₂CO₃ > 苯酚 > HCO₃⁻ |

## 第三节 醛 酮

### 醛 — 代表物：乙醛 CH₃CHO

**银镜反应**：
CH₃CHO + 2Ag(NH₃)₂OH →△→ CH₃COONH₄ + 2Ag↓ + 3NH₃ + H₂O

**与新制Cu(OH)₂反应**：
CH₃CHO + 2Cu(OH)₂ + NaOH →△→ CH₃COONa + Cu₂O↓(砖红色) + 3H₂O

**与H₂加成**：CH₃CHO + H₂ →Ni/△→ CH₃CH₂OH（还原反应）

**甲醛 HCHO 的特性**：
- 35%~40%水溶液 = 福尔马林（防腐）
- 1mol HCHO 与足量银氨溶液反应生成 4mol Ag（相当于含 2 个醛基）

### 酮 — 代表物：丙酮 CH₃COCH₃
- 不能发生银镜反应（无醛基）
- 与H₂加成 → CH₃CHOHCH₃（异丙醇）

---

## 第四节 羧酸 羧酸衍生物

### 羧酸 — 代表物：乙酸 CH₃COOH
- 弱酸，酸性 > H₂CO₃ > 苯酚
- 酯化反应：CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O
  - 酸脱羟基，醇脱氢（¹⁸O示踪法证实）

### 酯
- 通式 CnH₂nO₂（与羧酸互为同分异构体）
- **酸性水解**（可逆）：RCOOR' + H₂O ⇌ RCOOH + R'OH
- **碱性水解**（完全）：RCOOR' + NaOH → RCOONa + R'OH
- 油脂的碱性水解 = **皂化反应**

### 其他衍生物
- 酰胺：含—CONH₂ 官能团
- 胺：含—NH₂ 官能团，显碱性

---

## 第五节 有机合成

### 逆合成分析法
- 从目标分子出发，逆向推导合成路线
- 断键位置通常在官能团附近

### 碳链增长/缩短
- **增长**：R—X + NaCN → R—CN；格氏试剂反应；羟醛缩合
- **缩短**：脱羧反应（R—COONa + NaOH →CaO/△→ R—H + Na₂CO₃）

### 官能团转化
```
R—CH₃ → R—CH₂Cl → R—CH₂OH → R—CHO → R—COOH → R—COOR'
卤代      卤代烃     醇        醛       羧酸      酯
```

### 官能团的保护
- 酚羟基：先转化为—OCH₃，反应完成后再用HI还原
- 双键先加Br₂保护，反应完成后消去生成双键
