## 第一节 有机化合物的结构特点

### 有机物的成键特点
- 碳原子最外层4个电子，可形成4个共价键
- 可形成单键、双键、三键，可成链/成环
- 有机物的主要构成元素：C、H、O、N、S、P、X（卤素）

### 同分异构体

**构造异构**：
- 碳链异构（如正丁烷与异丁烷）
- 位置异构（如1-丁烯与2-丁烯）
- 官能团异构
  | 分子式 | 可能类别 |
  |--------|---------|
  | CnH₂n+₂O | 醇 / 醚 |
  | CnH₂nO | 醛 / 酮 / 烯醇/环醇 |
  | CnH₂nO₂ | 羧酸 / 酯 / 羟基醛 |

**立体异构**：
- 顺反异构（双键/环，不能旋转）
- 手性对映异构（手性碳）

### 不饱和度（Ω）— 缺氢指数 ★★

**定义**：以链状烷烃（CₙH₂ₙ₊₂）为饱和基准，每少 2 个 H → Ω=1

**计算通式**：Ω = (同碳数链状烷烃的H原子数 - 现有H原子数) / 2

| 化合物类型 | 公式 | 说明 |
|-----------|------|------|
| 烃及含氧/硫衍生物 | Ω = [(2n+2) - H数] / 2 | O、S 不影响 Ω |
| 卤代烃 | Ω = [(2n+2) - (H数 + X数)] / 2 | X → 当 H 算 |
| 含氮化合物 | Ω = [(2n+2) - (H数 - N数)] / 2 | 每个 N 引入 1 个 H |

**Ω 与结构的关系**：

| Ω | 可能结构 |
|----|---------|
| 1 | 1 个双键（C=C / C=O / C=N）或 1 个环 |
| 2 | 1 个三键 / 2 个双键 / 2 个环 / 环+双键 |
| 4 | 一般含苯环 |
| ≥4 | 一般含苯环 |

**注意**：
- 无论几元环，1 个环 Ω = 1
- 桥环化合物（如降冰片烷）Ω = 可断键数，不能直接数环
- 硝基（—NO₂）的 Ω = 1
- 苯环可简化为「3π + 1环」→ Ω = 4

**教学技巧**：
- Ω=4 **优先考虑苯环**，这是高中同分异构体题的核心突破口
- Ω 很大时，直接数 H 原子数比算 Ω 更省事——补全每个碳的 4 根键
- 链状烷烃 CnH₂ₙ₊₂ 是基准线，**必须记牢**

**Ω 的三大应用**：
1. **判断分子式正误**：算 Ω → 看结构 Ω → 是否相等
2. **书写分子式**：先确定非 H 原子数 → 数结构 Ω → 反推 H 数
3. **推同分异构体**：算 Ω → 推可能结构片段（双键/环/苯环）

> **常考**：含苯环芳香族化合物 Ω ≥ 4；如算出 Ω=3 则不可能含苯环

### 碳原子的分类
- 伯碳（1°）连1个碳、仲碳（2°）连2个碳、叔碳（3°）连3个碳、季碳（4°）连4个碳

### 有机物表示方法
- 分子式、结构式、结构简式、键线式、球棍模型、空间填充模型

## 第二节 研究有机化合物的一般方法

### 分离提纯
- 蒸馏：沸点不同的互溶液体
- 重结晶：利用温度对溶解度的影响不同
- 萃取：利用溶质在不同溶剂中溶解度不同
- 色谱法：利用吸附/分配能力不同

### 元素分析
- 燃烧法测定 C、H 含量（李比希法）

### 质谱
- 测定相对分子质量（最大质荷比峰 = 分子离子峰）

### 红外光谱（IR）
- 测定分子中**官能团 / 化学键**（不同化学键对红外吸收频率不同）

### 核磁共振氢谱（¹H NMR）
- 峰面积比 = 不同化学环境中氢原子数之比
- 化学位移：不同环境中H的出峰位置不同
- 峰的裂分数 = 相邻碳上H数 + 1（n+1规则）
