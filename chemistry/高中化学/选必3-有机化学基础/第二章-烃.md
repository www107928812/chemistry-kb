# 第二章 烃

> 人教版（2019）选择性必修3

## 第一节 烷烃

### 通式：CnH₂n+₂（n≥1）

### 甲烷 CH₄
- 正四面体，键角 109°28'，sp³ 杂化
- **取代反应**（与卤素在光照下）：CH₄ + Cl₂ →光→ CH₃Cl + HCl
- 高温分解：CH₄ →高温→ C + 2H₂

### 烷烃的化学性质
- 可燃性：CnH₂n+₂ + (3n+1)/2 O₂ →点燃→ nCO₂ + (n+1)H₂O
- 取代反应（光照）：与 Cl₂/Br₂
- 不能使酸性KMnO₄和溴水褪色

### 烷烃的系统命名
1. 选最长的碳链为主链（称某烷）
2. 从离支链最近的一端开始编号
3. 写名称：编号-取代基名称 + 某烷（不同取代基按次序规则排列）

## 第二节 烯烃 炔烃

### 烯烃通式：CnH₂n（n≥2，单烯烃）

### 乙烯 C₂H₄
- 平面结构，6原子共面，键角≈120°，C为sp²杂化
- 含 C=C（1个σ+1个π）
- 化学性质：
  - **加成**：CH₂=CH₂ + Br₂→CH₂BrCH₂Br（溴水褪色，检验C=C）
  - 与H₂/Ni：→ CH₃CH₃
  - 与HX：→ CH₃CH₂X
  - 与H₂O（催化剂）：→ CH₃CH₂OH
  - **加聚**：nCH₂=CH₂ → [-CH₂-CH₂-]n
  - **氧化**：燃烧（明亮火焰）/ 使酸性KMnO₄褪色

### 共轭二烯烃（1,3-丁二烯 CH₂=CH-CH=CH₂）
- 1,2-加成和1,4-加成

### 炔烃通式：CnH₂n-₂（n≥2）

### 乙炔 C₂H₂（HC≡CH）
- 直线形，4原子共线，C为sp杂化
- 含 C≡C（1个σ+2个π）
- 燃烧：2C₂H₂ + 5O₂ →点燃→ 4CO₂ + 2H₂O（**浓黑烟**）
- 氧炔焰温度达3000℃以上（焊接切割金属）
- 加成反应：加H₂（→C₂H₄→C₂H₆）、加Br₂（褪色）、加HX、加H₂O（→CH₃CHO，催化剂HgSO₄）
- 加聚：nCH≡CH → [-CH=CH-]n（聚乙炔，掺杂后导电）

#### 脂肪烃/卤代烃同分异构体书写方法

**总原则**：先算 Ω → 确定不饱和结构（双键/三键/环）→ 写碳骨架 → 安官能团

**步骤**：
1. **算 Ω**：确定是烷烃(Ω=0)、烯烃/环烷烃(Ω=1)、炔烃(Ω=2)还是卤代烃
2. **写碳骨架**：写出所有可能的碳链（直链/支链），按碳原子数从主链最长开始
3. **安官能团**：在每种碳骨架上移动官能团位置（双键/三键/卤原子）
4. **对称性去重**：检查每种结构的等效位置，排除重复结构

**卤代烃特别处理**：
- 计算 Ω 时，X 当 H 算
- 卤原子取代 H 的位置，同样遵循对称性防重复

**学生易错点**：
- ❌ 烯烃忘了还有环烷烃也是 Ω=1
- ❌ 碳骨架相同但忘了写支链位置编号
- ❌ 对称结构中重复位置没排除

## 第三节 芳香烃

### 苯 C₆H₆
- 平面正六边形，12原子共面，键角120°，C为sp²杂化
- 碳碳键介于单键和双键之间（大π键 Π₆⁶）
- **易取代，难加成**

**化学性质**：
1. 卤代：C₆H₆ + Br₂ →FeBr₃→ C₆H₅Br + HBr
2. 硝化：C₆H₆ + HNO₃ →浓H₂SO₄/50-60℃→ C₆H₅NO₂ + H₂O
3. 磺化：C₆H₆ + H₂SO₄(浓) → C₆H₅SO₃H + H₂O
4. 加成（难）：C₆H₆ + 3H₂ →Ni/△→ C₆H₁₂（环己烷）

#### 溴苯的制备与提纯（实验重点）

**反应**：C₆H₆ + Br₂ →FeBr₃→ C₆H₅Br + HBr↑

**反应后混合物**：溴苯、苯（过量）、Br₂、FeBr₃/Fe、HBr

| 物质 | 沸点 |
|------|------|
| 苯 | 80°C |
| 溴苯 | 156°C |

**提纯流程**（6步）：

```
混合物
  │
  ├── ① 过滤 ──→ 滤渣: Fe
  │
  └── 滤液
        │
        ├── ② 水洗、分液 ──→ 水层(弃): FeBr₃, HBr
        │
        └── 有机层
              │
              ├── ③ 碱洗、分液 ──→ 水层(弃): NaOH, NaBr, NaBrO₃
              │    （NaOH 除去未反应的 Br₂）
              └── 有机层
                    │
                    ├── ④ 水洗、分液 ──→ 水层(弃): NaOH 等
                    │    （洗去残留碱液）
                    └── 有机层（溴苯 + 苯）
                          │
                          ├── ⑤ CaCl₂ 干燥、过滤
                          │
                          └── 滤液
                                │
                                └── ⑥ 蒸馏 ──→ **溴苯**
                                     （利用沸点差，先蒸出苯）
```

**各步骤原理**：
- ① 过滤：除去铁粉（催化剂残留）
- ② 水洗：FeBr₃ 和 HBr 溶于水进入水层
- ③ 碱洗：Br₂ + 2NaOH → NaBr + NaBrO + H₂O，溴被碱液吸收
- ④ 水洗：洗去残留 NaOH
- ⑤ CaCl₂ 干燥：除去有机层中残留的水分
- ⑥ 蒸馏：苯沸点 80°C 先蒸出，溴苯 156°C 后收集

**学生易错点**：
- ❌ 忘记过量苯的存在，直接认为有机层只有溴苯
- ❌ 混淆各步水层和有机层分别含什么
- ❌ 碱洗目的不清楚（除 Br₂，不是除 HBr——HBr 在水洗那步已除去）

### 苯的同系物（甲苯 C₆H₅—CH₃）
- **能使酸性KMnO₄褪色**（侧链被氧化成—COOH）
- 苯不能使KMnO₄褪色 — 鉴别苯和甲苯！
- 甲苯的硝化（2,4,6-三硝基甲苯 = TNT）

### 定位效应
- 邻对位定位基（—CH₃、—OH、—NH₂、—X）→ 使新基团进入邻位和对位
- 间位定位基（—NO₂、—COOH、—CHO）→ 使新基团进入间位
