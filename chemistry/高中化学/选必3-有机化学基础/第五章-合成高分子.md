# 第五章 合成高分子

## 第一节 合成高分子的基本方法

### 加聚反应
- 含不饱和键的单体通过加成反应连接成高分子
- 例：nCH₂=CH₂ → [-CH₂—CH₂-]n（聚乙烯）
- 产物只有高分子，无小分子副产物

### 缩聚反应
- 含两个或两个以上官能团的单体缩合，脱去小分子（H₂O、HCl等）
- 例：nHO—R—COOH → [-O—R—CO-]n + nH₂O
- 形成聚酯、聚酰胺，脱去小分子

### 单体与聚合物的互推
- **加聚物 → 单体**：主链仅C—C单键，每两个C一组断开，恢复双键
- **缩聚物 → 单体**：找到断裂位置（酯基/酰胺键水解断开），补OH/H

## 第二节 高分子材料

### 塑料
| 塑料 | 单体 | 特点 |
|------|------|------|
| 聚乙烯 PE | CH₂=CH₂ | 热塑性 |
| 聚氯乙烯 PVC | CH₂=CHCl | 热塑性 |
| 聚丙烯 PP | CH₂=CHCH₃ | 热塑性 |
| 聚苯乙烯 PS | CH₂=CHC₆H₅ | 热塑性 |
| 聚四氟乙烯 PTFE | CF₂=CF₂ | 热塑性 |
| 酚醛塑料（电木） | 苯酚+甲醛 | **热固性** |

### 合成纤维
- 涤纶（聚酯纤维）— 对苯二甲酸 + 乙二醇
- 尼龙（聚酰胺纤维）
- 腈纶（聚丙烯腈）

### 合成橡胶
- 顺丁橡胶、丁苯橡胶、硅橡胶等

### 功能高分子
- 高吸水性树脂
- 离子交换树脂
- 导电高分子（如掺杂后的聚乙炔）

### 高分子的降解
- 白色污染：难降解塑料的积累
- 可降解塑料：聚乳酸(PLA)等
- 光降解、生物降解
