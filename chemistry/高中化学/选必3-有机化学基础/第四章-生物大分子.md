## 糖类

### 单糖
| | 葡萄糖 | 果糖 |
|----|--------|------|
| 分子式 | C₆H₁₂O₆ | C₆H₁₂O₆ |
| 结构 | 多羟基醛（含醛基） | 多羟基酮 |
| 关系 | 互为**同分异构体** | |
| 银镜反应 | **能** | 不能（无醛基） |
| 与Cu(OH)₂ | 砖红色沉淀 Cu₂O↓ | 不能 |

### 二糖
| | 蔗糖 | 麦芽糖 |
|----|------|--------|
| 分子式 | 均为 C₁₂H₂₂O₁₁，互为同分异构体 | |
| 还原性 | 无还原性 | 有还原性 |
| 水解产物 | 葡萄糖 + 果糖 | 2分子葡萄糖 |

### 多糖
| | 淀粉 | 纤维素 |
|----|------|--------|
| 通式 | 均为 (C₆H₁₀O₅)n | |
| 关系 | n值不同，**不是同分异构体** | |
| 特征反应 | 遇碘变**蓝色** | — |
| 水解最终产物 | **葡萄糖** | **葡萄糖** |

## 蛋白质

### 氨基酸
- 含—NH₂（氨基）和—COOH（羧基）
- 常见α-氨基酸：甘氨酸（氨基乙酸）、丙氨酸、苯丙氨酸
- 两性：既能与酸反应又能与碱反应
- 缩合：氨基酸 → 多肽（肽键 —CO—NH—）

### 蛋白质的性质
| 性质 | 条件 | 特点 |
|------|------|------|
| **盐析** | 加NaCl/(NH₄)₂SO₄ | 析出，加水可溶解（**可逆·物理变化**） |
| **变性** | 加热/重金属/酸/碱/酒精/紫外线 | 永久失活（**不可逆·化学变化**） |
| 显色反应 | 加浓硝酸加热 | 含苯环的蛋白质变**黄** |
| 水解 | 酸/碱/酶催化 | → α-氨基酸 |

### 酶
- 生物催化剂，绝大多数是蛋白质
- 高效性、专一性、条件温和

## 核酸

### 组成
- 核酸 → 核苷酸 → 磷酸 + 核糖/脱氧核糖 + 含氮碱基
- DNA：脱氧核糖核酸（双螺旋结构，碱基配对 A-T、C-G）
- RNA：核糖核酸
