# 第三章 铁 金属材料

> 人教版（2019）必修第一册

## 第一节 铁及其化合物

### 铁 Fe
- 纯铁银白色，常见铁中含碳
- 与 O₂、H₂O 共同作用生锈（电化学腐蚀）
- 与酸：Fe + 2H⁺ → Fe²⁺ + H₂↑（**生成Fe²⁺不是Fe³⁺！**）

### 铁离子

| | Fe²⁺（亚铁） | Fe³⁺（铁离子） |
|----|------------|-------------|
| 颜色 | 浅绿色 | 黄色/棕黄色 |
| 性质 | 还原性（易→Fe³⁺） | 氧化性（可→Fe²⁺） |
| 与NaOH | 白色→灰绿→红褐色 | 红褐色沉淀 |
| 与KSCN | 无现象 | **血红色**（检验Fe³⁺） |

### 铁三角转化
```
    氧化剂(Cl₂、HNO₃、H₂O₂)
Fe ─────────────────────────→ Fe³⁺
│       强氧化剂            ↑
│  (Cl₂、HNO₃)              │ 还原剂
↓                           │ (Fe、Cu、I⁻)
Fe²⁺ ←── 还原剂(Fe、Zn) ── Fe
```

- Fe²⁺ → Fe³⁺：加氧化剂（Cl₂/H₂O₂/HNO₃/KMnO₄酸性）
- Fe³⁺ → Fe²⁺：加还原剂（Fe粉/Cu粉/KI）

### Fe(OH)₂ 的制备
- 需在无氧条件下制备（Fe²⁺易被O₂氧化）
- 方法：煮沸NaOH赶O₂ + 液封（加一层植物油）隔绝空气

## 第二节 金属材料

### 铝 Al
- 银白色轻金属
- **两性**：既与酸又与碱反应
  - 2Al + 6HCl → 2AlCl₃ + 3H₂↑
  - 2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑
- 铝热反应：2Al + Fe₂O₃ →高温→ Al₂O₃ + 2Fe（焊接铁轨）

### Al₂O₃ 与 Al(OH)₃ 的两性
| 与酸 | 与碱 |
|------|------|
| Al₂O₃ + 6H⁺ → 2Al³⁺ + 3H₂O | Al₂O₃ + 2OH⁻ → 2AlO₂⁻ + H₂O |
| Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O | Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O |

### Al³⁺ ⇌ Al(OH)₃ ⇌ AlO₂⁻ 三角转化
- Al³⁺ + 3OH⁻ → Al(OH)₃↓
- Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O
- AlO₂⁻ + H⁺ + H₂O → Al(OH)₃↓

### 合金
- 硬度一般 > 纯金属，熔点一般 < 纯金属
- 铁合金（生铁/钢）、铝合金、铜合金（青铜/黄铜/白铜）
