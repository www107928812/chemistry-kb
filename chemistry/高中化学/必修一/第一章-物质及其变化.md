# 第一章 物质及其变化

> 人教版（2019）必修第一册 | 高中化学基石

## 绪言

化学是在**原子、分子水平**上研究物质的组成、结构、性质、转化及其应用的基础自然科学。

**发展简史**：
- 古代中国：陶瓷、冶金、火药、造纸、酿造——李时珍《本草纲目》、宋应星《天工开物》
- 1661年 波义耳提出元素概念 → 1774年 拉瓦锡提出氧化学说 → 1803年 道尔顿原子论 → 1869年 门捷列夫元素周期律
- 20世纪：原子结构奥秘揭示，化学分支学科建立
- 中国现代：1965年人工合成结晶牛胰岛素、合成核糖核酸、世界先进炼油全流程技术

**高中化学课程结构**：必修（第一册+第二册）+ 选择性必修（1化学反应原理+2物质结构与性质+3有机化学基础）

## 第一节 物质的分类及转化

### 物质分类
```
物质
├── 纯净物
│   ├── 单质 — 金属/非金属/稀有气体
│   └── 化合物
│       ├── 氧化物（金属氧化物/非金属氧化物/两性氧化物）
│       ├── 酸（按元数/强弱/含氧否/挥发性分类）
│       ├── 碱（强碱/弱碱）
│       └── 盐（正盐/酸式盐/碱式盐）
└── 混合物 — 溶液、胶体、浊液
```

### 分散系
| 分散系 | 粒子大小 | 特征 | 实例 |
|--------|---------|------|------|
| 溶液 | < 1 nm | 均一、稳定、不沉降 | 食盐水 |
| 胶体 | 1~100 nm | 均一、介稳、**丁达尔效应** | Fe(OH)₃胶体 |
| 浊液 | > 100 nm | 不均一、不稳定 | 泥水 |

- **丁达尔效应**：光通过胶体时产生光亮的"通路"，是鉴别胶体和溶液的常用方法

### 物质转化
- 单质、氧化物、酸、碱、盐之间的转化关系
- 复分解反应条件：沉淀/气体/水（弱电解质）

## 第二节 离子反应

### 电解质
| 类别 | 定义 | 举例 |
|------|------|------|
| 电解质 | 水溶液或熔融态能导电的**化合物** | 酸、碱、盐、金属氧化物 |
| 非电解质 | 两者都不能导电 | 蔗糖、酒精、CO₂ |
| 强电解质 | 完全电离 | 强酸、强碱、大多数盐 |
| 弱电解质 | 部分电离，存在电离平衡 | 弱酸、弱碱、水 |

### 离子反应条件
1. 生成沉淀
2. 生成气体
3. 生成弱电解质（水、弱酸、弱碱）
4. 发生氧化还原反应

### 离子方程式书写
1. 写出化学方程式
2. 可溶性强电解质拆成离子
3. 删除不反应的旁观离子
4. 检查原子守恒 + 电荷守恒

### 离子共存判断
**不能共存的情况**：
- 生成沉淀：Ag⁺+Cl⁻、Ba²⁺+SO₄²⁻、Fe³⁺+OH⁻
- 生成气体：H⁺ + CO₃²⁻/HCO₃⁻
- 生成弱电解质：H⁺ + OH⁻、H⁺ + CH₃COO⁻
- 发生氧化还原：Fe³⁺ + I⁻、MnO₄⁻/H⁺ + Cl⁻
- 双水解：Al³⁺ + HCO₃⁻（泡沫灭火器原理）

### 审题注意
- "无色溶液"→ 排除 Fe³⁺(黄)、Fe²⁺(浅绿)、Cu²⁺(蓝)、MnO₄⁻(紫)
- "酸性/碱性溶液"→ 加 H⁺/OH⁻ 条件

## 第三节 氧化还原反应

### 基本概念
| 术语 | 解释 |
|------|------|
| 氧化反应 | 物质**失**电子，化合价**升高** |
| 还原反应 | 物质**得**电子，化合价**降低** |
| 氧化剂 | **得**电子的物质（被还原） |
| 还原剂 | **失**电子的物质（被氧化） |

**口诀**：升失氧，降得还

### 与四大基本反应类型的关系
- 置换反应一定是氧化还原反应
- 复分解反应一定不是氧化还原反应
- 化合/分解反应可能是也可能不是

### 常见氧化剂
KMnO₄/H⁺、K₂Cr₂O₇/H⁺、浓H₂SO₄、HNO₃、Cl₂、Br₂、O₂、H₂O₂、Fe³⁺

### 常见还原剂
活泼金属(Na、Al、Zn、Fe)、C、CO、H₂、H₂S、KI、Fe²⁺、SO₃²⁻

### 氧化性/还原性强弱
- 金属活动性顺序 → 金属还原性递减
- 非金属性 → 非金属氧化性递减
- **强者先反应**原则

### 方程式配平
1. 标变价
2. 列升降（化合价升降数）
3. 求最小公倍数（使升降总数相等）
4. 配系数（先配变价元素，再观察配其他）
5. 检查（原子守恒 + 电荷守恒）
