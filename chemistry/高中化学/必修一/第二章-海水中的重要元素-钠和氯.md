# 第二章 海水中的重要元素——钠和氯

> 人教版（2019）必修第一册 | 高中元素化合物起点

## 第一节 钠及其化合物

### 钠 Na
- 银白色、质软（可用刀切）、密度小于水
- 保存在煤油中（隔绝空气和水）
- **与水反应**：2Na + 2H₂O → 2NaOH + H₂↑
  - 现象：**浮、熔、游、响、红**（钠浮水面→熔成银白色小球→四处游动→发出嘶嘶声响→酚酞变红）
- **与氧气**：
  - 常温：4Na + O₂ → 2Na₂O（白色）
  - 点燃：2Na + O₂ →🔥→ Na₂O₂（淡黄色，火焰黄色）
- **焰色反应**：黄色（物理变化）

### 钠的化合物

| 化合物 | 化学式 | 俗称 | 特征 |
|--------|--------|------|------|
| 氧化钠 | Na₂O | — | 白色，与H₂O→NaOH |
| 过氧化钠 | Na₂O₂ | — | **淡黄色**，供氧剂 2Na₂O₂+2CO₂→2Na₂CO₃+O₂ |
| 碳酸钠 | Na₂CO₃ | 纯碱/苏打 | 热稳定性好，溶液碱性强 |
| 碳酸氢钠 | NaHCO₃ | 小苏打 | 加热分解 2NaHCO₃→△→Na₂CO₃+H₂O+CO₂↑ |

### Na₂CO₃ vs NaHCO₃ 鉴别
| 方法 | Na₂CO₃ | NaHCO₃ |
|------|---------|---------|
| 加酸 | 逐滴才冒泡 | **立即**冒泡 |
| 加热 | 不分解 | 分解，生成气体使石灰水变浑浊 |
| 加CaCl₂ | 白色沉淀 | 不沉淀 |

## 第二节 氯及其化合物

### 氯气 Cl₂
- **黄绿色**气体，有毒，密度大于空气
- 实验室制法：MnO₂ + 4HCl(浓) →△→ MnCl₂ + Cl₂↑ + 2H₂O
- 与水：Cl₂ + H₂O ⇌ HCl + HClO
- 与碱（制漂白液）：Cl₂ + 2NaOH → NaCl + NaClO + H₂O
- **强氧化性**：
  - 2Fe + 3Cl₂ →🔥→ 2FeCl₃（注意：生成Fe³⁺！）
  - 2Na + Cl₂ →🔥→ 2NaCl
  - H₂ + Cl₂ →🔥→ 2HCl（苍白色火焰，瓶口白雾）

### 氯水成分
Cl₂、H₂O、HClO、H⁺、Cl⁻、ClO⁻、OH⁻（少量）— **三分子四离子**

- 氯水久置 → 稀盐酸（HClO见光分解 2HClO→光→2HCl+O₂↑）

### HClO（次氯酸）
- 弱酸、强氧化性、**漂白性**（氧化漂白，不可逆）、不稳定

### 漂白粉 / 漂白液
- 有效成分：Ca(ClO)₂ / NaClO
- 漂白原理：Ca(ClO)₂ + CO₂ + H₂O → CaCO₃↓ + 2HClO

### Cl⁻ 检验
- 加 AgNO₃ 溶液 + 稀硝酸 → **白色沉淀**不溶解

## 第三节 物质的量

| 物理量 | 符号 | 单位 | 公式 |
|--------|------|------|------|
| 物质的量 | n | mol | 核心物理量 |
| 摩尔质量 | M | g/mol | n = m/M |
| 气体摩尔体积 | Vm | L/mol | 标况(0℃,101kPa)下约 22.4 L/mol |
| 物质的量浓度 | c | mol/L | c = n/V(溶液) |
| 阿伏加德罗常数 | NA | mol⁻¹ | ≈ 6.02×10²³ |

### 溶液的配制
**步骤**：计算→称量/量取→溶解→冷却→转移→洗涤→定容→摇匀→装瓶

**误差分析**（以 c = n/V 分析）：
- 未冷却即转移 → V偏小 → c偏大
- 定容时加水过量 → V偏大 → c偏小
- 转移时有洒落 → n偏小 → c偏小
