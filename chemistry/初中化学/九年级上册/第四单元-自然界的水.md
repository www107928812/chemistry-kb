# 第四单元 自然界的水

> 人教版（2024）九年级上册

## 课题1 水资源及其利用

### 水资源
- 地球表面约 71% 被水覆盖，但淡水资源仅约 2.53%
- 水污染来源：工业废水、农业农药化肥、生活污水

### 水的净化
| 方法 | 除去杂质类型 |
|------|-------------|
| 沉淀 | 不溶性固体（密度较大） |
| 过滤 | 不溶性固体（所有） |
| 吸附（活性炭） | 色素、异味、部分可溶性杂质 |
| 蒸馏 | 可溶性杂质（最彻底，得到纯水） |

**净化程度**：蒸馏 > 吸附 > 过滤 > 沉淀

- **硬水**：含较多 Ca²⁺、Mg²⁺；**软水**：含较少或不含
- 硬水鉴别：加肥皂水 → 泡沫少、浮渣多为硬水
- 硬水软化：煮沸（生活）/ 蒸馏（实验）
- **硬水危害**：浪费肥皂、衣物变硬、锅炉结垢（严重可引发爆炸）

### 过滤操作要点
- **一贴二低三靠**：滤纸紧贴漏斗；滤纸边缘低于漏斗、液面低于滤纸边缘；烧杯口靠玻璃棒、玻璃棒靠三层滤纸、漏斗下端靠烧杯内壁
- 过滤只能除去不溶性杂质，不能除去可溶性杂质

### 蒸馏注意事项
- 蒸馏烧瓶垫石棉网（受热均匀）；液体量 1/3~2/3；加沸石防暴沸；冷凝水逆流（下进上出）
- 蒸馏是净化程度最高的方法，蒸馏水属于纯净物

### 水体富营养化
- 氮、磷含量过高 → 藻类疯长 → 赤潮（海洋）/ 水华（湖泊）

## 课题2 水的组成

### 电解水实验
- 正极：产生 O₂（能使带火星木条复燃）
- 负极：产生 H₂（能被点燃，淡蓝色火焰）
- **体积比 O₂ : H₂ = 1 : 2**
- 结论：水由氢元素和氧元素组成

### 氢气
- 无色无味，密度最小，可燃
- 点燃前必须**验纯**（不纯会爆炸）
- 燃烧：2H₂ + O₂ →点燃→ 2H₂O
- 还原性：H₂ + CuO →△→ Cu + H₂O

### 电解水实验误差分析
- 实际 O₂:H₂ 体积比往往**小于** 1:2
- 原因：O₂ 在水中的溶解性比 H₂ 稍大；O₂ 氧化性强，可能在电极上发生反应消耗

## 课题3 物质组成的表示

### 化合价口诀（初中必背）
```
一价钾钠氯氢银（K⁺ Na⁺ Cl⁻ H⁺ Ag⁺）
二价氧钙钡镁锌（O²⁻ Ca²⁺ Ba²⁺ Mg²⁺ Zn²⁺）
三铝四硅五价磷（Al³⁺ Si⁴⁺ P⁵⁺）
二三铁，二四碳（Fe²⁺/Fe³⁺，C²⁺/C⁴⁺）
一价铵根硝酸根（NH₄⁺ NO₃⁻）
二价硫酸碳酸根（SO₄²⁻ CO₃²⁻）
```
- **规律**：化合物中各元素化合价代数和为 0

### 常见原子团（根）
OH⁻（氢氧根）、NO₃⁻（硝酸根）、SO₄²⁻（硫酸根）、CO₃²⁻（碳酸根）、NH₄⁺（铵根）、PO₄³⁻（磷酸根）

### 化学式的计算
- 相对分子质量 = 各原子相对原子质量之和
- 元素质量比
- 元素质量分数 = 该元素原子总质量 / 相对分子质量 × 100%

### 化合价与化学式
- 正价在前，负价在后
- 化合价代数和为零
- 原子个数取最小公倍数
