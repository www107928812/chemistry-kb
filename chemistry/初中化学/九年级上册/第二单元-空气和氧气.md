# 第二单元 空气和氧气

> 人教版（2024）九年级上册 | 中考高频考点

## 课题1 我们周围的空气

### 空气成分
| 成分 | 体积分数 | 性质 |
|------|---------|------|
| N₂ | ≈78% | 化学性质稳定 |
| O₂ | ≈21% | 支持燃烧、供给呼吸 |
| 稀有气体 | 0.94% | 化学性质非常稳定（通电发光→霓虹灯） |
| CO₂ | 0.03% | 能使澄清石灰水变浑浊 |

- **拉瓦锡测定空气成分实验**：汞在密闭容器中加热 12 天→银白色液态汞变红色粉末（HgO），空气体积减少约 1/5；将红色粉末加强热→重新得到汞和 O₂，O₂ 体积恰好等于之前减少的体积→得出结论：空气由 O₂ 和 N₂ 组成
- **红磷燃烧法**（课本实验 2-1）：红磷 + 氧气 →点燃→ 五氧化二磷（4P + 5O₂ → 2P₂O₅）。集气瓶内水面上升约 1/5 → O₂ 约占空气体积 1/5
- 稀有气体曾被称为"惰性气体"（过去认为不反应）

### 红磷燃烧实验误差分析
| 偏差 | 原因 |
|------|------|
| 偏低 | 红磷量不足；装置漏气；未冷却到室温就打开止水夹 |
| 偏高 | 点燃红磷后未立即伸入瓶中（瓶内空气受热逸出）；未夹紧弹簧夹（瓶内空气受热逸出） |

> **口诀**：偏低三因——量不足、漏气、未冷却

### 空气污染与防治
- **污染物**：SO₂、NO₂、CO、O₃、可吸入颗粒物（PM10/PM2.5）
- **酸雨成因**：SO₂、NO₂ 排放
- **温室效应**：CO₂ 过量排放
- **注意**：CO₂ 不是空气污染物，但含量增多会引起温室效应；空气污染指数越高，污染越严重
- **防治**：使用清洁能源、植树造林、汽车尾气净化、工厂废气处理

## 课题2 氧气

### 物理性质
- 无色无味气体，密度略大于空气，不易溶于水

### 化学性质（氧化性）
| 反应 | 化学方程式 | 现象 |
|------|-----------|------|
| 与碳 | C + O₂ →点燃→ CO₂ | 空气中红光，氧气中白光 |
| 与硫 | S + O₂ →点燃→ SO₂ | 空气中淡蓝色火焰，氧气中明亮蓝紫色火焰 |
| 与铁 | 3Fe + 2O₂ →点燃→ Fe₃O₄ | 剧烈燃烧，火星四射，生成黑色固体 |
| 与镁 | 2Mg + O₂ →点燃→ 2MgO | 耀眼白光 |
| 与磷 | 4P + 5O₂ →点燃→ 2P₂O₅ | 产生大量白烟 |

**O₂ 的检验**：带火星木条伸入集气瓶，木条复燃。
**验满**：向上排空气法→带火星木条放瓶口；排水法→瓶口有大气泡冒出。

### 氧化反应与缓慢氧化
- **氧化反应**：物质与氧发生的反应（"氧"不一定是 O₂，也可以是含氧化合物）
- **缓慢氧化**：进行很慢、不易察觉的氧化反应，如动植物的呼吸、食物腐烂、酿酒酿醋、农家肥腐熟
- O₂ 在氧化反应中提供氧，具有**氧化性**（助燃性 ≠ 可燃性）

> **学生误区**：氧气支持燃烧（助燃性），但氧气本身不可燃，不能做燃料

> **注意**：铁燃烧实验需在瓶底放少量水或细沙（防高温熔融物溅落炸裂瓶底）。

## 课题3 制取氧气

### 实验室制法

| 方法 | 化学方程式 | 装置类型 |
|------|-----------|---------|
| KMnO₄ 加热 | 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑ | 固+固→气 |
| KClO₃ + MnO₂ | 2KClO₃ →MnO₂/△→ 2KCl + 3O₂↑ | 固+固→气 |
| H₂O₂ + MnO₂ | 2H₂O₂ →MnO₂→ 2H₂O + O₂↑ | 固+液→气 |

### 收集方法
- **排水法**：O₂ 不易溶于水（较纯净）
- **向上排空气法**：O₂ 密度大于空气

### 催化剂（触媒）
- 改变反应速率，但本身质量和化学性质在反应后不变
- **一变两不变**：改变速率（可加快也可减慢），质量不变，化学性质不变
- MnO₂ 是 H₂O₂ 分解的催化剂；同一反应可能有多种催化剂（如 CuSO₄ 溶液也可催化 H₂O₂）
- 催化剂有选择性，只针对特定反应；没有催化剂反应仍可发生（只是速率不同）

### 工业制氧气
- **分离液态空气法**（利用各组分沸点不同）
- 属于**物理变化**（没有新物质生成）
- 氧气贮存在蓝色钢瓶中

### 排水法 vs 排空气法
| 方法 | 优点 | 缺点 |
|------|------|------|
| 排水法 | 收集气体**更纯净** | 气体含水分，不干燥 |
| 向上排空气法 | 收集气体**更干燥** | 纯度不如排水法 |

> 排空气法收集时导管要伸到接近集气瓶底部，便于排尽空气。

### 反应类型
- **化合反应**：A + B → C（多变一）
- **分解反应**：C → A + B（一变多）

## 实验活动1：氧气的实验室制取与性质

**制取步骤（KMnO₄ 法）**：
查（检查气密性）→ 装（装入药品，试管口塞棉花）→ 定（固定装置，试管口略向下倾斜）→ 点（点燃酒精灯加热）→ 收（排水法收集）→ 离（导管先移出水面）→ 熄（再熄灭酒精灯）

> **口诀**：茶（查）庄（装）定（定）点（点）收（收）利（离）息（熄）

## 中考考法归纳（基于真题分析）

### 考法1：燃烧条件与灭火原理辨析

**出题方式**：给说法/实验→判断燃烧相关概念正误（Q3/Q32）。

**燃烧三条件**：可燃物+氧气(或空气)+温度达着火点（三者同时满足，缺一不可）

**典型陷阱**（Q3火药）：
- "火药燃烧属于缓慢氧化"✗（燃烧是剧烈氧化）
- "有充足氧气就能燃烧"✗（还需温度达着火点）
- "火药燃烧一定引发爆炸"✗（有限空间内急速燃烧才爆炸）
- 火药需密封低温干燥储存✓

### 考法2：高低蜡烛燃烧探究

**出题方式**：密闭容器中高低蜡烛燃烧→传感器数据→判断说法（Q32）。

**关键结论**：
- CO₂密度大于空气→下沉→低处蜡烛先灭
- 气体受热密度变小→CO₂/有毒气体上升聚集高处
- 火场逃生→**弯腰沿墙根快速撤离**（低处氧气更充足、有毒气体少）
- 容器内CO₂含量<O₂含量（空气中N₂占78%！）

### 考法3：H₂O₂制O₂实验综合

**出题方式**：给4个制氧实验方案→找错误（Q14）。

**关键点**：
- H₂O₂+MnO₂制O₂→固液不加热，不需酒精灯✗
- 排水法收集→O₂不易溶于水✓
- 向上排空气法验满→带火星木条放瓶口复燃✓
- 铁丝在O₂中燃烧→验证助燃性✓

### 考法4：化学方程式正误

**出题方式**：给生活场景方程式→判断哪个不正确（Q26）。

**经典错误**：
- "酸雨：CO₂+H₂O→H₂CO₃"✗（酸雨是SO₂/NO₂→H₂SO₄/HNO₃，不是CO₂！）
- 铝制品耐腐蚀：4Al+3O₂→2Al₂O₃✓
- 自热米饭：CaO+H₂O→Ca(OH)₂✓
- 光合作用：6CO₂+6H₂O→C₆H₁₂O₆+6O₂✓

### 考法5：光合作用与物质循环

**出题方式**：结合生物知识→考查O₂/CO₂循环（Q46）。

金鱼藻+斑马鱼：金鱼藻光合作用→O₂供斑马鱼呼吸；斑马鱼呼吸→CO₂供金鱼藻光合。物质循环闭环。
