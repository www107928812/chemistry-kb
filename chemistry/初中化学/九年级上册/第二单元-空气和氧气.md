# 第二单元 空气和氧气

> 人教版（2024）九年级上册 | 中考高频考点

## 课题1 我们周围的空气

### 空气成分
| 成分 | 体积分数 | 性质 |
|------|---------|------|
| N₂ | ≈78% | 化学性质稳定 |
| O₂ | ≈21% | 支持燃烧、供给呼吸 |
| 稀有气体 | 0.94% | 化学性质非常稳定（通电发光→霓虹灯） |
| CO₂ | 0.03% | 能使澄清石灰水变浑浊 |

- **拉瓦锡测定空气成分实验**：汞在密闭容器中加热 12 天→银白色液态汞变红色粉末（HgO），空气体积减少约 1/5；将红色粉末加强热→重新得到汞和 O₂，O₂ 体积恰好等于之前减少的体积→得出结论：空气由 O₂ 和 N₂ 组成
- **红磷燃烧法**（课本实验 2-1）：红磷 + 氧气 →点燃→ 五氧化二磷（4P + 5O₂ → 2P₂O₅）。集气瓶内水面上升约 1/5 → O₂ 约占空气体积 1/5
- 稀有气体曾被称为"惰性气体"（过去认为不反应）

### 空气污染
- 有害气体：CO、SO₂、NO₂、O₃
- PM2.5/PM10（可吸入颗粒物）
- 酸雨成因：SO₂、NO₂ 排放
- 温室效应：CO₂ 过量排放

## 课题2 氧气

### 物理性质
- 无色无味气体，密度略大于空气，不易溶于水

### 化学性质（氧化性）
| 反应 | 化学方程式 | 现象 |
|------|-----------|------|
| 与碳 | C + O₂ →点燃→ CO₂ | 空气中红光，氧气中白光 |
| 与硫 | S + O₂ →点燃→ SO₂ | 空气中淡蓝色火焰，氧气中明亮蓝紫色火焰 |
| 与铁 | 3Fe + 2O₂ →点燃→ Fe₃O₄ | 剧烈燃烧，火星四射，生成黑色固体 |
| 与镁 | 2Mg + O₂ →点燃→ 2MgO | 耀眼白光 |
| 与磷 | 4P + 5O₂ →点燃→ 2P₂O₅ | 产生大量白烟 |

**O₂ 的检验**：带火星木条伸入集气瓶，木条复燃。

> **注意**：铁燃烧实验需在瓶底放少量水或细沙（防高温熔融物溅落炸裂瓶底）。

## 课题3 制取氧气

### 实验室制法

| 方法 | 化学方程式 | 装置类型 |
|------|-----------|---------|
| KMnO₄ 加热 | 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑ | 固+固→气 |
| KClO₃ + MnO₂ | 2KClO₃ →MnO₂/△→ 2KCl + 3O₂↑ | 固+固→气 |
| H₂O₂ + MnO₂ | 2H₂O₂ →MnO₂→ 2H₂O + O₂↑ | 固+液→气 |

### 收集方法
- **排水法**：O₂ 不易溶于水（较纯净）
- **向上排空气法**：O₂ 密度大于空气

### 催化剂（触媒）
- 改变反应速率，但本身质量和化学性质在反应后不变
- MnO₂ 是 H₂O₂ 分解的催化剂

### 反应类型
- **化合反应**：A + B → C（多变一）
- **分解反应**：C → A + B（一变多）

## 实验活动1：氧气的实验室制取与性质

**制取步骤（KMnO₄ 法）**：
查（检查气密性）→ 装（装入药品，试管口塞棉花）→ 定（固定装置，试管口略向下倾斜）→ 点（点燃酒精灯加热）→ 收（排水法收集）→ 离（导管先移出水面）→ 熄（再熄灭酒精灯）

> **口诀**：茶（查）庄（装）定（定）点（点）收（收）利（离）息（熄）
