# 第五单元 化学反应的定量关系

> 人教版（2024）九年级上册 | 中考核心考点（质量守恒+方程式计算）

## 课题1 质量守恒定律

### 内容
**参加化学反应的各物质的质量总和 = 反应后生成的各物质的质量总和**

### 微观解释
反应前后：原子种类不变、原子数目不变、原子质量不变。因此反应物总质量 = 生成物总质量。

### 应用
1. 解释化学反应中物质质量变化
2. 推断某反应物或生成物的化学式
3. 计算反应物或生成物的质量

### 注意事项
- 只适用于化学变化，不适用于物理变化
- "参加反应的"——剩余的不算
- 有气体参与/生成的反应，须在密闭容器中验证

### 测定实验（红磷燃烧验证质量守恒）
- 装置：锥形瓶 + 气球（缓冲气压）
- 气球先膨胀后缩小（红磷燃烧放热→气体受热膨胀→冷却后瓶内气压减小）

## 课题2 化学方程式

### 意义
1. 表示反应物和生成物
2. 表示各物质间的分子个数比
3. 表示各物质间的质量比

### 书写步骤
1. 写出反应物和生成物的化学式
2. 配平（最小公倍数法）
3. 标注反应条件（加热△、点燃、催化剂等）
4. 标注生成物状态（↑气体 ↓沉淀）

### 配平方法
- **最小公倍数法**：以出现次数较少、原子数较复杂的元素入手
- **观察法**：先配平金属元素→非金属元素→氧元素→氢元素

### 四大基本反应类型

| 类型 | 通式 | 特征 |
|------|------|------|
| 化合反应 | A + B → C | 多变一 |
| 分解反应 | AB → A + B | 一变多 |
| 置换反应 | A + BC → B + AC | 单质+化合物→新单质+新化合物 |
| 复分解反应 | AB + CD → AD + CB | 互换成分，需生成沉淀/气体/水 |

**注意**：CO₂ + Ca(OH)₂ → CaCO₃↓ + H₂O 不属于四种基本反应类型。

### 化学方程式计算步骤
1. 写出化学方程式并配平
2. 找出已知量和未知量，标注相对分子质量
3. 列比例式
4. 求解
5. 写答案

```
例：10g CaCO₃ 完全分解可得多少克 CaO？

CaCO₃ →△→ CaO + CO₂↑
100        56
10         x

100/10 = 56/x  →  x = 5.6g
```
