# 第六单元 碳和碳的氧化物

> 人教版（2024）九年级上册 | 中考必考（CO₂ 制取是实验探究核心）

### "碳"与"炭"区别
- **碳**：元素名称，如碳元素、碳单质、二氧化碳
- **炭**：具体物质名称，如木炭、活性炭、焦炭、炭黑

### 无定形碳
| 类型 | 制法 | 用途 |
|------|------|------|
| 木炭 | 木材隔绝空气加强热 | 燃料、冶炼金属、制活性炭 |
| 活性炭 | 木炭高温水蒸气处理 | 净水、防毒面具（吸附是物理性质） |
| 焦炭 | 煤隔绝空气加强热（干馏） | 冶炼金属 |
| 炭黑 | 含碳物质不完全燃烧 | 墨、油墨、橡胶填料 |

## 课题1 碳单质的多样性

### 同素异形体
| | 金刚石 | 石墨 | C₆₀（足球烯） |
|----|--------|------|-------------|
| 结构 | 正四面体网状 | 层状 | 球状分子 |
| 硬度 | 最硬 | 质软 | — |
| 导电性 | 不导电 | 导电 | — |
| 用途 | 切割玻璃、做钻头 | 电极、润滑剂 | 超导材料 |

- **无定形碳**：活性炭（吸附性——物理性质）、木炭、焦炭、炭黑
- 同素异形体：由同种元素组成的不同单质（物理性质不同，化学性质相同）

### 碳的化学性质
1. **稳定性**（常温）：古代字画保存至今
2. **可燃性**：
   - 充分燃烧：C + O₂ →点燃→ CO₂
   - 不充分燃烧：2C + O₂ →点燃→ 2CO
3. **还原性**：
   - C + 2CuO →高温→ 2Cu + CO₂↑（黑色固体变红色）
   - C + CO₂ →高温→ 2CO

## 课题2 碳的氧化物

### CO₂
| 物理性质 | 化学性质 |
|----------|----------|
| 无色无味，密度>空气，能溶于水 | 不支持燃烧，与水/碱反应 |

- **检验**：通入澄清石灰水 → 变浑浊 Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O
- **吸收**：NaOH 溶液 2NaOH + CO₂ → Na₂CO₃ + H₂O
- **与水**：CO₂ + H₂O → H₂CO₃（使紫色石蕊变红的是 H₂CO₃，不是 CO₂！）
- 固体 CO₂ 称"干冰"，用于人工降雨、制冷（升华吸热，物理变化）
- **不用于燃烧**、不供给呼吸

**CO₂ 对环境的影响**：温室效应

### CO
| 物理性质 | 化学性质 |
|----------|----------|
| 无色无味，密度略小于空气，难溶于水 | 毒性、可燃性、还原性 |

- **毒性**：与血红蛋白结合（比O₂能力强），造成人体缺氧
- **可燃性**：2CO + O₂ →点燃→ 2CO₂（**蓝色火焰**）
- **还原性**：CO + CuO →△→ Cu + CO₂（黑色→红色，检验生成的CO₂）

> **CO 与 CO₂ 的关系**：CO → CO₂（燃烧/还原CuO）；CO₂ → CO（与C高温反应）

## 课题3 二氧化碳的实验室制取

### 原理
- **CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑**
- 石灰石/大理石（主要成分 CaCO₃）+ 稀盐酸

### 药品选用
- 不能用稀硫酸代替稀盐酸：生成 CaSO₄（微溶）附着在CaCO₃表面，阻碍反应
- 不能用浓盐酸：HCl 挥发混入 CO₂
- 不能用 Na₂CO₃ 代替 CaCO₃：反应速率太快，不易控制

### 装置
- 发生装置：固+液→气（与 H₂O₂ 制 O₂ 相同）
- 收集装置：向上排空气法（不能用排水法——CO₂ 能溶于水）

### 验满
- 燃着的木条放在瓶口，木条熄灭则满

### 除杂
- 先通过 NaHCO₃ 溶液（除 HCl）
- 再用浓 H₂SO₄（除水蒸气）

## 实验活动3：二氧化碳的实验室制取与性质
**CO₂ 性质实验**：
- 阶梯蜡烛实验 → 下层蜡烛先灭 → 说明 CO₂ 密度>空气且不支持燃烧

## 中考考法归纳（基于真题分析）

### 考法1：CO₂制取装置评价

**出题方式**：给CO₂制取+用途装置→判断正误（Q20）。

**水草缸CO₂补给装置**：
- 甲装置(固液不加热)→不能控制反应速率（需分液漏斗才能控速）
- 乙瓶水→不可用NaOH溶液代替（NaOH会吸收CO₂）
- 导管伸入水草缸→置于底部③处（CO₂密度>水，利于与水草接触）
- 通CO₂目的→光合作用原料（非调节酸碱度）

### 考法2：CO₂性质验证实验

**出题方式**：干燥CO₂通入含湿润/干燥石蕊纸花的装置→判断说法（Q48）。

**实验设计分析**：
- 浓硫酸干燥CO₂→可行✓
- 变红顺序：c→b→a（CO₂密度>空气，先沉底→先接触下方湿润纸花）
- 干燥纸花a不变红→CO₂本身不酸
- 湿润纸花变红→CO₂+H₂O→H₂CO₃（碳酸使石蕊变红）

**陷阱**：变红顺序不是a-c-b!

### 考法3：微观反应示意图（CO₂相关）

**出题方式**：CO₂参与的微观反应→判断反应类型/分子个数比（Q31/Q60）。

**水+CO₂制乙烯**（Q31）：
- 反应Ⅰ:H₂O→H₂+O₂(分解反应)
- 反应Ⅱ:CO₂+H₂→C₂H₄+H₂O(非置换!生成物均为化合物)
- 总过程涉及2种氧化物(H₂O、CO₂)
- 反应Ⅰ耗水>反应Ⅱ产水→需补水

**CO₂+NH₃制尿素**（Q60）：
- 步骤①:CO₂+2NH₃→NH₂COONH₄(化合,分子比1:2)
- 步骤②:NH₂COONH₄→CO(NH₂)₂+H₂O(分解,原子守恒)
- 总方程式:CO₂+2NH₃→CO(NH₂)₂+H₂O

### 考法4：实验现象描述

**出题方式**：燃烧/反应现象描述→找正确选项（Q15）。

- 硫在O₂中→明亮蓝紫色火焰✓
- 浓H₂SO₄敞口→无白雾（浓盐酸才白雾）
- 碘难溶于水→紫红色溶液✗（碘溶于酒精才呈紫红色）
