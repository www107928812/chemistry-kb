# 第一单元 走进化学世界

> 人教版（2024）九年级上册 | 对应中考基础选择题

## 绪论 化学使世界变得更加绚丽多彩

### 化学的定义
化学是研究物质的**组成、结构、性质、转化及应用**的一门基础学科，特征是从**分子层次**认识物质，通过**化学变化**创造物质。

### 化学发展简史
| 时期 | 代表人物/事件 | 贡献 |
|------|-------------|------|
| 远古 | 人类学会用火 | 最早的化学实践 |
| 古代中国 | 陶瓷、冶金、酿造、造纸、火药 | 多项化学工艺世界领先 |
| 近代 | 拉瓦锡（法） | 发现质量守恒定律 |
| 近代 | 道尔顿（英） | 提出原子论 |
| 近代 | 阿伏加德罗（意） | 提出分子学说 |
| 近代 | 门捷列夫（俄） | 发现元素周期律 |
| 现代 | 中国科学家 | 合成结晶牛胰岛素、创建稀土分离理论 |

> **中考常考**：道尔顿→原子论；阿伏加德罗→分子学说；门捷列夫→元素周期律

### 化学与生活
- 衣：合成纤维、染料
- 食：化肥、农药
- 住：水泥、钢材、玻璃、涂料
- 行：汽油、柴油等燃料
- 环保：绿色化学（绿色、低碳、循环发展）

## 课题1 物质的变化和性质

### 物理变化 vs 化学变化

| | 物理变化 | 化学变化 |
|----|----------|----------|
| 定义 | 没有新物质生成 | 有新物质生成 |
| 伴随现象 | 形状、状态改变 | 发光、放热、变色、生成气体/沉淀|
| 本质 | 分子本身不变 | 分子破裂，原子重新组合 |
| 举例 | 水结冰、铁丝弯曲、糖溶于水 | 铁生锈、蜡烛燃烧 |

**常见易错判断**：
- 灯泡发光 → 物理变化（无新物质）
- 石油分馏 → 物理变化（沸点不同）
- 煤的干馏 → 化学变化
- 蒸馏 vs 干馏：蒸馏是物理变化，干馏是化学变化

### 物理性质 vs 化学性质

| 物理性质 | 化学性质 |
|----------|----------|
| 颜色、状态、气味、密度、熔点、沸点、硬度、溶解性、导电性、导热性 | 可燃性、氧化性、还原性、酸碱性、稳定性 |

**性质与用途的关系**：性质决定用途，用途体现性质。例：石墨导电→做电极。

### 物质分类

| 分类 | 定义 | 举例 |
|------|------|------|
| **混合物** | 由多种物质组成 | 空气、海水、矿泉水、煤、石油、天然气 |
| **纯净物** | 由一种物质组成 | 液氧、液氮、冰水混合物、金刚石、氯化钠、水 |
| **单质** | 由一种元素组成的纯净物 | 金刚石、液氧、铁、氧气、氢气 |
| **化合物** | 由不同元素组成的纯净物 | 氯化钠、冰水混合物、干冰 |

> **学生误区**：冰水混合物是纯净物（只有H₂O一种物质），不是混合物

## 课题2 化学实验与科学探究

### 常见实验仪器

| 仪器 | 用途 | 注意事项 |
|------|------|---------|
| 试管 | 少量试剂反应 | 外壁擦干，管口不对人，液体不超1/3 |
| 烧杯 | 配制溶液、较多量反应 | 垫石棉网加热 |
| 量筒 | 粗略量取液体体积 | 读数时视线与凹液面最低处相平 |
| 托盘天平 | 称量固体质量 | 左物右码，精确到0.1g |
| 酒精灯 | 加热 | 外焰温度最高，用灯帽盖灭（不能用嘴吹） |
| 漏斗 | 过滤 | 一贴二低三靠 |
| 蒸发皿 | 蒸发浓缩 | 直接加热，用坩埚钳取 |
| 胶头滴管 | 吸取和滴加少量试剂 | 用后清洗；取液时先排空气 |
| 玻璃棒 | 搅拌、引流、蘸取液体 | 搅拌时不碰容器内壁 |
| 铁架台 | 固定和支持仪器 | 保持干燥，防酸碱腐蚀 |
| 集气瓶 | 收集或贮存少量气体 | 不能加热；燃烧实验瓶底铺细沙或水防炸裂 |
| 广口瓶 | 储存固体药品 | — |
| 细口瓶 | 储存液体药品 | 瓶塞倒放桌面防污染 |
| 坩埚钳 | 夹持坩埚或加热物 | — |
| 燃烧匙 | 支持可燃固体燃烧 | 防酸碱腐蚀 |
| 药匙 | 取粉末状药品 | 用后擦净 |
| 镊子 | 取块状药品 | 用后擦净 |
| 长颈漏斗 | 向反应容器注入液体 | 制气体时下端管口插入液面以下（液封） |

### 仪器加热分类
- **可直接加热**：试管、蒸发皿、坩埚、燃烧匙
- **垫石棉网加热**：烧杯、烧瓶、锥形瓶
- **不可加热**：量筒、集气瓶、漏斗

### 量筒读数误差
- **仰视**：读数偏小 → 实际液体体积偏大
- **俯视**：读数偏大 → 实际液体体积偏小
- 口诀：**"仰小实大，俯大实小"**

### 药品取用
**三原则**：
- 三不：不触、不闻、不尝
- 节约：未说明用量时，液体 1~2mL，固体盖满试管底部
- 处理：剩余药品不放回原瓶、不随意丢弃、放入指定容器

**固体取用**：
- 块状：一平二竖三慢立（镊子夹取，沿管壁滑入）
- 粉末：一斜二送三直立（药匙/纸槽送入底部）

**液体取用**：标签向手心，瓶口紧挨试管口，瓶塞倒放桌面
**闻气味**：用手在瓶口轻轻扇动

### 物质的加热
- 固体加热：试管口略向下倾斜（防冷凝水倒流炸裂）
- 液体加热：试管口略向上倾斜45°，管口不对人，液体不超1/3
- 酒精灯：酒精量 1/4~2/3；外焰温度最高；灯帽盖灭，不可嘴吹；洒出燃烧用湿抹布盖灭

### 托盘天平
- 左物右码，精确度 0.1g
- 一般药品两盘各垫一张等大纸；腐蚀性药品（如NaOH）放玻璃器皿中称量
- 砝码用镊子夹取，先大后小，最后移游码

### 连接仪器与气密性检查
- 玻璃管插入橡皮塞：先用水润湿，转动插入
- 气密性检查：导管一端入水，双手紧贴容器外壁 → 导管口有气泡冒出（松手后形成一段水柱），则气密性好

### 玻璃仪器洗涤
- 油污：洗涤剂或热纯碱溶液
- 难溶性碳酸盐/碱：稀盐酸浸泡后冲洗
- 洗净标准：内壁附着的水既不聚成水滴，也不成股流下

### 常见危险化学品标志
- 易燃气体/液体/固体、氧化剂、腐蚀品、有毒品、爆炸品等

### 意外处理
- 酸沾皮肤：大量水冲洗→涂 3%~5% NaHCO₃
- 碱沾皮肤：大量水冲洗→涂硼酸溶液
- 酒精灯失火：湿抹布盖灭
- 浓硫酸溅到皮肤：先干布擦，再大量水冲洗→涂NaHCO₃

## 中考考法归纳（基于15道真题分析）

### 考法1：物理变化 vs 化学变化的辨析

**出题方式**：给4个生活/生产场景，判断哪个是化学变化（或物理变化）。

**典型题干特征**：
- 直接判断型：`下列变化属于化学变化的是（）`
- 诗词俗语型：`下列诗句或俗语中，涉及化学反应的是（）`（Q5）
- 古文语境型：`下列语言描述不涉及化学性质的是（）`（Q3）

**出题人常用素材**：
| 物理变化（干扰项） | 化学变化（正确项） |
|------|------|
| 海水晒盐、风力发电、铁水铸锅（Q2） | 树叶腐烂（Q2） |
| 吹尽黄沙始见金、酒香不怕巷子深、铁杵磨成绣花针（Q5） | 爝火燃回春浩浩（Q5） |
| 凡海盐，见水即化（Q3） | 熬胆矾铁釜，久之亦化为铜（Q3） |

**核心区分标准**：有无**新物质**生成。发光、放热、变色、气体、沉淀是辅助判断线索，但不是充要条件。

### 考法2："性质" vs "变化" 的文字陷阱

**这是第一单元错误率最高的题型**，初中生极易混淆。

**出题方式**：给出一组描述，要求分别归类为"物理变化/化学变化/物理性质/化学性质"。

**核心判断口诀**：
- 描述中有 **"能/可以/会/容易"** → **性质**（描述能力/属性）
- 描述中有 **"正在/已经/变成了"** → **变化**（描述事件/过程）

**Q1示例**：`A．酒精能挥发 B．铁在潮湿的空气里生成铁锈 C．水沸腾变成水蒸气 D．二氧化碳能使澄清的石灰水变浑浊`

| 选项 | 归类 | 理由 |
|------|------|------|
| A．酒精**能**挥发 | 物理性质 | "能"=能力，挥发是物理属性 |
| B．铁在潮湿空气里**生成**铁锈 | 化学变化 | "生成"=产生新物质 |
| C．水沸腾**变成**水蒸气 | 物理变化 | "变成"=状态改变，无新物质 |
| D．二氧化碳**能使**石灰水变浑浊 | 化学性质 | "能使"=能力，通过化学反应体现 |

> **学生最易错点**：把A（酒精能挥发）误判为物理变化。判断标准——"能挥发"说的是**挥发性**（能力/属性），不是正在发生的挥发事件。

**Q7同类题**：`下列对一氧化碳性质的描述，属于化学性质的是（）`
- A/B/C 均为物理性质（颜色、水溶性、密度）
- D 可燃性 → 化学性质

> **出题人套路**：用同一种物质的多种性质做选项（全是一氧化碳），考"物理性质 vs 化学性质"的分类。

### 考法3：古诗文/传统文化情境

**近年中考趋势**，把化学概念藏在传统文化表述中。

**Q3（古文）**：`凡海盐，见水即化` → "化"不是化学变化，是溶解（物理性质）

**Q5（诗词）**：
- `吹尽黄沙始见金` → 物理分离
- `爝火燃回春浩浩` → 燃烧（化学变化）
- `酒香不怕巷子深` → 分子扩散（物理变化）
- `铁杵磨成绣花针` → 形状改变（物理变化）

> **教学提示**：让学生积累常见古诗文对应的化学原理，这是中考新宠。

### 考法4：情境信息提取

**Q4（雷雨发庄稼）**：给一段科普材料→从材料中提取指定信息（物理性质）。

**出题模式**：材料描述系列化学反应→问题问的是物理性质（或其他基础信息），学生容易因材料复杂而慌乱。

**解题思路**：
1. 定位关键词：问题问的是"物理性质"
2. 到材料中找描述性词语（不涉及"生成了什么"）
3. "无色、气体、难溶于水"是典型的物理性质描述

> **教学提示**：题目长≠题目难。让学生先看问题再回去定位，不要被材料吓住。

### 考法5：实验仪器——用途匹配

**Q8（仪器字母匹配）**：给出仪器图片和字母→填写对应仪器用途的字母。
**Q9（仪器用途选择）**：`可用于配制溶液、加热较多量液体及反应容器的是（）` → 烧杯

**必须过关的仪器用途对应表**（中考高频，每题必有一道）：

| 问题场景 | 答案仪器 |
|------|------|
| 吸取和滴加少量液体 | 胶头滴管 |
| 配制溶液、加热较多液体 | 烧杯 |
| 量取一定体积液体 | 量筒 |
| 实验室常用加热仪器 | 酒精灯 |
| 少量试剂反应容器 | 试管 |
| 过滤 | 漏斗 |

### 考法6：实验操作正误判断

**第一单元分值最重的题型**，Q10-Q12 + Q15 共4题涉及。

**出题方式**：
- 纯文字判断题（Q10：20个操作逐一判断）
- 单图选择题（Q11、Q12：四选一找错误/正确操作）
- 多题合并（Q15：多道小题合并，含滴管操作、仪器用途、闻气味、验满、取用粉末等）

**20个操作中错误频率最高的考点**（Q10答案分析）：

| 操作 | 要点 |
|------|------|
| 加热液体 | 液体不超试管容积1/3，管口不对人 |
| 称量固体 | 左物右码 |
| 倾倒液体 | 瓶口紧挨、瓶塞倒放、标签向手心 |
| 点燃酒精灯 | 禁止用燃着的酒精灯引燃另一盏 |
| 量筒读数 | 平视凹液面最低处 |
| 取粉末药品 | 药匙或纸槽，一斜二送三直立 |
| 加入块状固体 | 一平二竖三慢立，沿管壁滑入 |
| 塞橡皮塞 | 慢慢转动塞进，不可硬塞 |
| 振荡试管 | 手腕左右摆动，不是上下晃动 |
| 固体药品保存 | 广口瓶（不是细口瓶） |
| 量筒用途 | 不能溶解药品、不能加热、不能作反应器 |

### 考法7：量筒读数误差——仰视俯视

**这是第一单元的"压轴陷阱"**，Q13+Q14连续两题都在考。

**基本规律**（Q13）：
- 仰视 → 读数偏小 → 实际液体偏多
- 俯视 → 读数偏大 → 实际液体偏少
- 口诀：**"羊（仰）小虎（俯）大"** → 注意这是**读数**，实际体积恰好相反

**进阶陷阱**（Q14）：`先仰视读19mL，倒出部分后再俯视读10mL，实际倒出多少？`

这是**连环套**：
1. 仰视读19mL → 实际液体**多于**19mL
2. 俯视读10mL → 实际剩余液体**少于**10mL
3. 倒出量 = 实际原量 - 实际余量 **>** 19-10 = 9mL
4. 答案：**大于9mL**

> **学生最容易错**：直接19-10=9，忽略了两次读数误差的**叠加效应**。两次误差方向一致（都让实际值偏离读数值），导致实际差值更大。

**出题人思路**：这题不止考"仰小俯大"的结论，更考学生能否把误差**代入两步操作**中推理。属于区分度较高的题目。

### 考法8：性质与用途错配判断

**出题方式**：给4组"性质-用途"配对，找不一致的一个。

**核心公式**：性质决定用途，用途体现性质。

**典型陷阱**（Q5）：
| 物质 | 性质 | 用途 | 是否一致 |
|------|------|------|----------|
| 醋酸 | 酸性 | 去厨房油污 | ✗（酸性不去油污→碱性去油污）|
| NaCl | 咸味 | 调味品 | ✓ |
| 活性炭 | 吸附性 | 冰箱除臭 | ✓ |
| 天然气 | 可燃性 | 燃料 | ✓ |

> **教学提示**：酸可以去水垢(碳酸盐)但不能去油污(油脂)。去油污靠碱性物质(Na₂CO₃/NaOH)。

### 考法9：实验安全图标识别

**出题方式**：给一张图含多个安全图标→选出某类实验必须使用的图标。

**常考图标**（Q38）：
| 图标 | 含义 | 适用场景 |
|------|------|---------|
| 护目镜 | 防液体飞溅 | 金属与酸反应、浓酸稀释 |
| 手套 | 防腐蚀 | 取用腐蚀性药品 |
| 实验服 | 保护身体 | 有腐蚀性/毒害的化学实验 |
| 排气扇 | 通风 | 产生有害气体的实验 |

> **学生误区**：金属与稀硫酸反应需要护目镜+手套+实验服，但不需要排气扇（H₂无毒）。

### 考法10：垃圾分类中的化学标志

**出题方式**：给废旧物品→判断应投入哪个垃圾桶（可回收/有害/厨余/其他）。

**关键原则**（Q1）：空矿泉水塑料瓶→**可回收物**（有回收利用价值，塑料可再造）。

**常考对应**：
| 垃圾 | 类别 | 标志特征 |
|------|------|---------|
| 塑料瓶、废纸、金属 | 可回收 | 三角形循环箭头 |
| 废电池、过期药品 | 有害垃圾 | ×形标志 |
| 剩饭剩菜 | 厨余垃圾 | 食物图案 |
| 砖瓦陶瓷、渣土 | 其他垃圾 | 箭头向下三角形 |

### 考法11：实验操作正误判断（综合图判）

**出题方式**：4个实验操作图→判断不正确的是哪个。

**高频错误操作**（Q45）：
| 操作 | 正确要点 | 常见错误 |
|------|----------|----------|
| 稀释浓硫酸 | 酸入水+搅拌 | 水入酸（会沸腾溅出）|
| 浓硫酸干燥气体 | 长进短出 | 接反（短进长出无法充分干燥）|
| **过滤** | **玻璃棒引流+漏斗下端贴烧杯壁** | **无引流+下端悬空（最高频！）** |
| 验证金属活动性 | 前金置换后金 | 选错金属组合 |

### 考法12：实验现象描述规范

**出题方式**：给4个实验现象描述→判断哪个正确。

**核心禁区**（Q15）：
- **不能出现生成物名称**：描述"生成红色物质"而非"生成红色的铜"
- **碘溶于水呈紫红色？→错**：碘难溶于水，碘溶于酒精才呈紫红色
- **浓硫酸敞口→白雾？→错**：浓硫酸吸水性，不产生白雾（浓盐酸才产生白雾）
- **硫在氧气中燃烧→蓝紫色火焰✓**
