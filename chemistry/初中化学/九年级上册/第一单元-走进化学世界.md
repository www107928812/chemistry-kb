# 第一单元 走进化学世界

> 人教版（2024）九年级上册 | 对应中考基础选择题

## 绪论 化学使世界变得更加绚丽多彩

### 化学的定义
化学是研究物质的**组成、结构、性质、转化及应用**的一门基础学科，特征是从**分子层次**认识物质，通过**化学变化**创造物质。

### 化学发展简史
| 时期 | 代表人物/事件 | 贡献 |
|------|-------------|------|
| 远古 | 人类学会用火 | 最早的化学实践 |
| 古代中国 | 陶瓷、冶金、酿造、造纸、火药 | 多项化学工艺世界领先 |
| 近代 | 拉瓦锡（法） | 发现质量守恒定律 |
| 近代 | 道尔顿（英） | 提出原子论 |
| 近代 | 阿伏加德罗（意） | 提出分子学说 |
| 近代 | 门捷列夫（俄） | 发现元素周期律 |
| 现代 | 中国科学家 | 合成结晶牛胰岛素、创建稀土分离理论 |

> **中考常考**：道尔顿→原子论；阿伏加德罗→分子学说；门捷列夫→元素周期律

### 化学与生活
- 衣：合成纤维、染料
- 食：化肥、农药
- 住：水泥、钢材、玻璃、涂料
- 行：汽油、柴油等燃料
- 环保：绿色化学（绿色、低碳、循环发展）

## 课题1 物质的变化和性质

### 物理变化 vs 化学变化

| | 物理变化 | 化学变化 |
|----|----------|----------|
| 定义 | 没有新物质生成 | 有新物质生成 |
| 伴随现象 | 形状、状态改变 | 发光、放热、变色、生成气体/沉淀|
| 本质 | 分子本身不变 | 分子破裂，原子重新组合 |
| 举例 | 水结冰、铁丝弯曲、糖溶于水 | 铁生锈、蜡烛燃烧 |

**常见易错判断**：
- 灯泡发光 → 物理变化（无新物质）
- 石油分馏 → 物理变化（沸点不同）
- 煤的干馏 → 化学变化
- 蒸馏 vs 干馏：蒸馏是物理变化，干馏是化学变化

### 物理性质 vs 化学性质

| 物理性质 | 化学性质 |
|----------|----------|
| 颜色、状态、气味、密度、熔点、沸点、硬度、溶解性、导电性、导热性 | 可燃性、氧化性、还原性、酸碱性、稳定性 |

**性质与用途的关系**：性质决定用途，用途体现性质。例：石墨导电→做电极。

### 物质分类

| 分类 | 定义 | 举例 |
|------|------|------|
| **混合物** | 由多种物质组成 | 空气、海水、矿泉水、煤、石油、天然气 |
| **纯净物** | 由一种物质组成 | 液氧、液氮、冰水混合物、金刚石、氯化钠、水 |
| **单质** | 由一种元素组成的纯净物 | 金刚石、液氧、铁、氧气、氢气 |
| **化合物** | 由不同元素组成的纯净物 | 氯化钠、冰水混合物、干冰 |

> **学生误区**：冰水混合物是纯净物（只有H₂O一种物质），不是混合物

## 课题2 化学实验与科学探究

### 常见实验仪器

| 仪器 | 用途 | 注意事项 |
|------|------|---------|
| 试管 | 少量试剂反应 | 外壁擦干，管口不对人，液体不超1/3 |
| 烧杯 | 配制溶液、较多量反应 | 垫石棉网加热 |
| 量筒 | 粗略量取液体体积 | 读数时视线与凹液面最低处相平 |
| 托盘天平 | 称量固体质量 | 左物右码，精确到0.1g |
| 酒精灯 | 加热 | 外焰温度最高，用灯帽盖灭（不能用嘴吹） |
| 漏斗 | 过滤 | 一贴二低三靠 |
| 蒸发皿 | 蒸发浓缩 | 直接加热，用坩埚钳取 |
| 胶头滴管 | 吸取和滴加少量试剂 | 用后清洗；取液时先排空气 |
| 玻璃棒 | 搅拌、引流、蘸取液体 | 搅拌时不碰容器内壁 |
| 铁架台 | 固定和支持仪器 | 保持干燥，防酸碱腐蚀 |
| 集气瓶 | 收集或贮存少量气体 | 不能加热；燃烧实验瓶底铺细沙或水防炸裂 |
| 广口瓶 | 储存固体药品 | — |
| 细口瓶 | 储存液体药品 | 瓶塞倒放桌面防污染 |
| 坩埚钳 | 夹持坩埚或加热物 | — |
| 燃烧匙 | 支持可燃固体燃烧 | 防酸碱腐蚀 |
| 药匙 | 取粉末状药品 | 用后擦净 |
| 镊子 | 取块状药品 | 用后擦净 |
| 长颈漏斗 | 向反应容器注入液体 | 制气体时下端管口插入液面以下（液封） |

### 仪器加热分类
- **可直接加热**：试管、蒸发皿、坩埚、燃烧匙
- **垫石棉网加热**：烧杯、烧瓶、锥形瓶
- **不可加热**：量筒、集气瓶、漏斗

### 量筒读数误差
- **仰视**：读数偏小 → 实际液体体积偏大
- **俯视**：读数偏大 → 实际液体体积偏小
- 口诀：**"仰小实大，俯大实小"**

### 药品取用
**三原则**：
- 三不：不触、不闻、不尝
- 节约：未说明用量时，液体 1~2mL，固体盖满试管底部
- 处理：剩余药品不放回原瓶、不随意丢弃、放入指定容器

**固体取用**：
- 块状：一平二竖三慢立（镊子夹取，沿管壁滑入）
- 粉末：一斜二送三直立（药匙/纸槽送入底部）

**液体取用**：标签向手心，瓶口紧挨试管口，瓶塞倒放桌面
**闻气味**：用手在瓶口轻轻扇动

### 物质的加热
- 固体加热：试管口略向下倾斜（防冷凝水倒流炸裂）
- 液体加热：试管口略向上倾斜45°，管口不对人，液体不超1/3
- 酒精灯：酒精量 1/4~2/3；外焰温度最高；灯帽盖灭，不可嘴吹；洒出燃烧用湿抹布盖灭

### 托盘天平
- 左物右码，精确度 0.1g
- 一般药品两盘各垫一张等大纸；腐蚀性药品（如NaOH）放玻璃器皿中称量
- 砝码用镊子夹取，先大后小，最后移游码

### 连接仪器与气密性检查
- 玻璃管插入橡皮塞：先用水润湿，转动插入
- 气密性检查：导管一端入水，双手紧贴容器外壁 → 导管口有气泡冒出（松手后形成一段水柱），则气密性好

### 玻璃仪器洗涤
- 油污：洗涤剂或热纯碱溶液
- 难溶性碳酸盐/碱：稀盐酸浸泡后冲洗
- 洗净标准：内壁附着的水既不聚成水滴，也不成股流下

### 常见危险化学品标志
- 易燃气体/液体/固体、氧化剂、腐蚀品、有毒品、爆炸品等

### 意外处理
- 酸沾皮肤：大量水冲洗→涂 3%~5% NaHCO₃
- 碱沾皮肤：大量水冲洗→涂硼酸溶液
- 酒精灯失火：湿抹布盖灭
- 浓硫酸溅到皮肤：先干布擦，再大量水冲洗→涂NaHCO₃
