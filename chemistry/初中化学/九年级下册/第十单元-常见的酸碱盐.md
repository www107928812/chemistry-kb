# 第十单元 常见的酸、碱、盐

> 人教版（2024）九年级下册 | 中考最重要板块（推断题/实验探究题核心）

## 课题1 溶液的酸碱性

### 酸碱指示剂

| 指示剂 | 酸性 | 中性 | 碱性 |
|--------|------|------|------|
| 紫色石蕊 | 红 | 紫 | 蓝 |
| 无色酚酞 | 无 | 无 | 红 |

### pH 值
- pH < 7 → **酸性**（越小越强）
- pH = 7 → **中性**
- pH > 7 → **碱性**（越大越强）
- 测定：pH 试纸（整数，不能润湿！）、pH 计

## 课题2 常见的酸和碱

### 常见酸

| 酸 | 化学式 | 特性 |
|----|--------|------|
| 盐酸 | HCl | 无色，有挥发性（白雾） |
| 硫酸 | H₂SO₄ | 浓硫酸吸水/脱水/强氧化 |
| 硝酸 | HNO₃ | 浓硝酸有强氧化性 |

**浓硫酸稀释**：酸入水，沿器壁缓慢加入，不断搅拌（千万不能水入酸！）

### 酸的化学通性（五点）
1. 与指示剂：石蕊变红，酚酞不变色
2. 与活泼金属（H之前）→ 盐 + H₂↑
3. 与金属氧化物 → 盐 + H₂O
4. 与碱 → 盐 + H₂O（**中和反应**）
5. 与某些盐 → 新盐 + 新酸（复分解）

### 常见碱

| 碱 | 化学式 | 俗称 | 特性 |
|----|--------|------|------|
| 氢氧化钠 | NaOH | 烧碱/火碱/苛性钠 | 白色固体，易潮解，强腐蚀 |
| 氢氧化钙 | Ca(OH)₂ | 熟石灰/消石灰 | 微溶，石灰水 |

### 碱的化学通性（四点）
1. 与指示剂：石蕊变蓝，酚酞变红
2. 与非金属氧化物 → 盐 + H₂O
3. 与酸 → 盐 + H₂O（中和反应）
4. 与某些盐 → 新盐 + 新碱

> **NaOH 敞口放置**：吸水潮解（物理变化）+ 与 CO₂ 反应 2NaOH + CO₂ → Na₂CO₃ + H₂O（化学变化）

## 课题3 常见的盐

### 常见盐

| 盐 | 化学式 | 俗名/用途 |
|----|--------|----------|
| 氯化钠 | NaCl | 食盐（调味、防腐、生理盐水） |
| 碳酸钠 | Na₂CO₃ | 纯碱/苏打（制玻璃、造纸） |
| 碳酸氢钠 | NaHCO₃ | 小苏打（发酵粉、治疗胃酸过多） |
| 碳酸钙 | CaCO₃ | 石灰石/大理石（建筑材料、补钙剂） |
| 硫酸铜 | CuSO₄·5H₂O | 蓝矾/胆矾 |

### 复分解反应的条件
生成物中有**沉淀**或**气体**或**水**

### 常见沉淀（必记）

| 颜色 | 沉淀 |
|------|------|
| 白色 | CaCO₃、BaCO₃、AgCl、BaSO₄、Mg(OH)₂、Al(OH)₃ |
| 蓝色 | Cu(OH)₂ |
| 红褐色 | Fe(OH)₃ |
| 不溶于酸的白色 | **AgCl、BaSO₄** |

### 粗盐提纯
步骤：溶解 → 过滤 → 蒸发
除可溶杂质顺序：BaCl₂（除SO₄²⁻）→ NaOH（除Mg²⁺）→ Na₂CO₃（除Ca²⁺+多余Ba²⁺）→ 过滤 → 盐酸调pH

### 化肥
- **氮肥**：CO(NH₂)₂ 尿素（含氮量最高）、NH₄NO₃
- **磷肥**：Ca₃(PO₄)₂
- **钾肥**：KCl、K₂SO₄
- **复合肥**：KNO₃、NH₄H₂PO₄
