# 第十单元 常见的酸、碱、盐

> 人教版（2024）九年级下册 | 中考最重要板块（推断题/实验探究题核心）

## 课题1 溶液的酸碱性

### 酸碱指示剂

| 指示剂 | 酸性 | 中性 | 碱性 |
|--------|------|------|------|
| 紫色石蕊 | 红 | 紫 | 蓝 |
| 无色酚酞 | 无 | 无 | 红 |

### pH 值
- pH < 7 → **酸性**（越小越强）
- pH = 7 → **中性**
- pH > 7 → **碱性**（越大越强）
- 测定：pH 试纸（整数，不能润湿！）、pH 计

## 课题2 常见的酸和碱

### 常见酸

| 酸 | 化学式 | 特性 |
|----|--------|------|
| 盐酸 | HCl | 无色，有挥发性（白雾） |
| 硫酸 | H₂SO₄ | 浓硫酸吸水/脱水/强氧化 |
| 硝酸 | HNO₃ | 浓硝酸有强氧化性 |

**浓硫酸稀释**：酸入水，沿器壁缓慢加入，不断搅拌（千万不能水入酸！）

### 酸的化学通性（五点）
1. 与指示剂：石蕊变红，酚酞不变色
2. 与活泼金属（H之前）→ 盐 + H₂↑
3. 与金属氧化物 → 盐 + H₂O
4. 与碱 → 盐 + H₂O（**中和反应**）
5. 与某些盐 → 新盐 + 新酸（复分解）

### 常见碱

| 碱 | 化学式 | 俗称 | 特性 |
|----|--------|------|------|
| 氢氧化钠 | NaOH | 烧碱/火碱/苛性钠 | 白色固体，易潮解，强腐蚀 |
| 氢氧化钙 | Ca(OH)₂ | 熟石灰/消石灰 | 微溶，石灰水 |

### 碱的化学通性（四点）
1. 与指示剂：石蕊变蓝，酚酞变红
2. 与非金属氧化物 → 盐 + H₂O
3. 与酸 → 盐 + H₂O（中和反应）
4. 与某些盐 → 新盐 + 新碱

> **NaOH 敞口放置**：吸水潮解（物理变化）+ 与 CO₂ 反应 2NaOH + CO₂ → Na₂CO₃ + H₂O（化学变化）

## 课题3 常见的盐

### 常见盐

| 盐 | 化学式 | 俗名/用途 |
|----|--------|----------|
| 氯化钠 | NaCl | 食盐（调味、防腐、生理盐水） |
| 碳酸钠 | Na₂CO₃ | 纯碱/苏打（制玻璃、造纸） |
| 碳酸氢钠 | NaHCO₃ | 小苏打（发酵粉、治疗胃酸过多） |
| 碳酸钙 | CaCO₃ | 石灰石/大理石（建筑材料、补钙剂） |
| 硫酸铜 | CuSO₄·5H₂O | 蓝矾/胆矾 |

### 复分解反应的条件
生成物中有**沉淀**或**气体**或**水**

### 常见沉淀（必记）

| 颜色 | 沉淀 |
|------|------|
| 白色 | CaCO₃、BaCO₃、AgCl、BaSO₄、Mg(OH)₂、Al(OH)₃ |
| 蓝色 | Cu(OH)₂ |
| 红褐色 | Fe(OH)₃ |
| 不溶于酸的白色 | **AgCl、BaSO₄** |

### 粗盐提纯
步骤：溶解 → 过滤 → 蒸发
除可溶杂质顺序：BaCl₂（除SO₄²⁻）→ NaOH（除Mg²⁺）→ Na₂CO₃（除Ca²⁺+多余Ba²⁺）→ 过滤 → 盐酸调pH

### 化肥
- **氮肥**：CO(NH₂)₂ 尿素（含氮量最高）、NH₄NO₃
- **磷肥**：Ca₃(PO₄)₂
- **钾肥**：KCl、K₂SO₄
- **复合肥**：KNO₃、NH₄H₂PO₄

## 中考考法归纳（基于真题分析）

### 考法1：中和反应滴定曲线

**出题方式**：给pH-时间/温度曲线→判断溶液中离子/温度变化（Q35）。

**读数技巧**：
- pH<7→酸过量（H⁺多）；pH>7→碱过量（OH⁻多）；pH=7→恰好完全
- 温度最高点=恰好完全反应（中和放热）
- **M点前离子个数不变**：H⁺减少但Na⁺等量增加，总数不变

### 考法2：pH+自制指示剂综合

**出题方式**：给pH试纸数据+自制指示剂变色表→综合判断（Q39）。

**紫薯汁/紫甘蓝汁作指示剂**：
- 酸红碱绿（中性紫）
- 草木灰水变绿→显碱性pH>7
- 酸稀释pH↑(酸性减弱)，碱稀释pH↓(碱性减弱)

### 考法3：波尔多液配制

**出题方式**：围绕CuSO₄+Ca(OH)₂→CaSO₄+Cu(OH)₂↓设置选项（Q27）。

**关键点**：
- 生石灰+水→Ca(OH)₂（放热）→再与CuSO₄反应→蓝色沉淀Cu(OH)₂
- **不能铁制容器配波尔多液**：Fe+CuSO₄→FeSO₄+Cu
- 主要成分：Cu(OH)₂(沉淀)+CaSO₄

### 考法4：中和反应概念辨析

**出题方式**：给一组酸+碱反应→判断概念正误（Q8）。

**核心陷阱**：
- 生成盐和水的反应不一定是中和反应（CO₂+2NaOH→Na₂CO₃+H₂O）
- 中和反应特指酸+碱→盐+水

### 考法5：物质推断网络（框图推断）

**出题方式**：A-F六种物质，部分已知条件→推断并回答问题（Q53）。

**解题策略**：
1. 先锁定"确定性最强"的物质（如D=H₂O是溶剂/氧化物）
2. 利用"配制波尔多液"→B=CuSO₄(盐)+E=Ca(OH)₂(碱)
3. 利用"五种不同类别"→锁定剩余A(单质/酸)+C(剩余类)
4. 利用"F是钠盐"→F=Na₂CO₃
5. 逐步验证所有转化关系→判断选项

### 考法6：粗盐精制（第十单元核心实验）

**出题方式**：给粗盐提纯流程图→判断操作/试剂/产物正误（Q41）。

**除杂试剂+顺序**：
```
滤液1(NaCl+MgCl₂+Na₂SO₄+CaCl₂)
 → +过量NaOH(除Mg²⁺)
 → +过量BaCl₂(除SO₄²⁻,试剂甲)
 → +过量Na₂CO₃(除Ca²⁺+多余Ba²⁺)
 → 过滤(白色滤渣4种:Mg(OH)₂+BaSO₄+CaCO₃+BaCO₃)
 → 滤液2 → +过量稀盐酸(除OH⁻/CO₃²⁻)
 → 蒸发结晶 → 纯净NaCl
```

> **顺序不能乱**：Na₂CO₃必须在BaCl₂之后（否则多余Ba²⁺除不掉）！

### 考法7：物质推断（转化网络）

**出题方式**：甲/乙/丙/丁+CO₂转化网络,给条件→逐项判断（Q59）。

**关键条件**："甲乙丁至少含一种相同元素"限制了解题方向。

**典型代入法**：
- A:甲=O₂→丁=CO(还原性✓)
- B:丙/丁黑=C/CuO→乙=CO→CO+CuO→Cu+CO₂非置换✗
- D:乙/丙含Ca→CaO+CO₂/Ca(OH)₂+Na₂CO₃→复分解可能✓

### 考法8：酸碱改良+肥料选择

**出题方式**：给土壤pH→选合适肥料（Q51）。

**双重要求题**：
- pH=4酸性→需碱性物质中和→草木灰(K₂CO₃水解碱性)✓
- 增强抗病虫抗倒伏→钾肥作用✓
- NaOH不能用于土壤(强腐蚀性)✗

**肥料功能口诀**：氮促叶、磷促花/根/抗寒、钾增强抗病虫抗倒伏

### 考法9：传统文化中的酸碱盐

**出题方式**：古籍记载→用化学知识解析（Q55）。

《本草纲目》"冬月灶中烧薪柴之灰，以灰淋汁，取碱浣衣"：
- "烧"→化学变化(燃烧)✓
- "灰淋汁"→溶解(水溶出K₂CO₃)✓
- "灰"含K₂CO₃✓
- "浣衣"→利用碱性去油污(非强腐蚀性)✗

### 考法10：实验方案评价

**出题方式**：给探究目的+方案→判断是否可行（Q52）。

**除杂铁律**：不能引入新杂质，不能破坏目标物质。
- CaO中除CaCO₃加稀盐酸→两者都反应✗（应高温煅烧）
- MnO₂中除KCl→溶解过滤(利用溶解性差异)✓
