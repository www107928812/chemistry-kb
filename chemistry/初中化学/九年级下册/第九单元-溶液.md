# 第九单元 溶液

> 人教版（2024）九年级下册 | 中考必考（溶解度曲线+溶质质量分数计算）

## 课题1 溶液及其应用

### 溶液的定义
- **溶液**：一种或几种物质分散到另一种物质里形成的均一、稳定的混合物
- 溶质 + 溶剂 → 溶液
- 特征：**均一性、稳定性**、混合物

### 乳化作用
- **乳化**：使不溶于水的液体分散成小液滴，形成乳浊液（不均一不稳定）
- 洗涤剂去油污 → 乳化作用（物理变化）；汽油去油污 → 溶解（物理变化）；NaOH去油污 → 化学反应
- **注意区分**：溶解、乳化、化学反应三种去油污原理

### 溶质与溶剂的判断
- 固体/气体 + 液体 → 固体/气体是溶质，液体是溶剂
- 两种液体互溶 → 量多的为溶剂（但只要有水，水永远是溶剂）
- 命名：溶质的溶剂溶液，如碘的酒精溶液（碘酒）

### 溶解现象
- 物质溶解时常伴随**温度变化**：
  - NaOH 溶于水 → 温度**升高**
  - NH₄NO₃ 溶于水 → 温度**降低**
  - NaCl 溶于水 → 温度变化不大

## 课题2 溶解度

### 饱和溶液与不饱和溶液
- **饱和溶液**：在一定温度下、一定量溶剂中，不能再溶解某种溶质的溶液
- 相互转化（大多数固体）：
  ```
  不饱和 ⇌ 饱和
    加溶质/降温/蒸发溶剂 →
    ← 加溶剂/升温
  ```

### 固体溶解度
- **定义**：在一定温度下，某固体在 100g 溶剂里达到饱和状态时所溶解的质量（单位：g）
- 大多数固体溶解度随温度升高而**增大**（KNO₃ 型）
- NaCl 溶解度几乎不受温度影响
- **Ca(OH)₂ 溶解度随温度升高反而降低**（特殊）

### 气体溶解度
- 随温度升高而减小，随压强增大而增大
- 喝汽水打嗝（温度↑→气体溶解度↓）、开汽水瓶冒泡（压强↓→溶解度↓）

### 溶解度曲线图读图（中考必考）
- 交点 → 该温度下两种物质溶解度相等
- 曲线上方 → 饱和溶液 + 有未溶固体
- 分离方法：
  - KNO₃（陡升型）→ **降温结晶**（冷却热饱和溶液）
  - NaCl（缓升型）→ **蒸发结晶**

## 课题3 溶质的质量分数

### 基本公式
- **ω = m(溶质) / m(溶液) × 100%**
- m(溶液) = m(溶质) + m(溶剂)
- 饱和溶液中：ω = S / (S+100) × 100%（S 为溶解度）

### 溶液稀释
- 稀释前后**溶质质量不变**
- m₁ω₁ = m₂ω₂

### 配制一定溶质质量分数的溶液
**步骤**：计算 → 称量（托盘天平称固体 / 量筒量液体）→ 溶解（烧杯 + 玻璃棒搅拌）→ 装瓶贴标签

**误差分析**：
- 称量时砝码和药品放反（用游码）→ 偏小
- 量取水时仰视读数 → 水偏多 → 偏小
- 转移时有洒落 → 偏小

## 中考考法归纳（基于真题分析）

### 考法1：溶解度曲线读图+计算

**这是第九单元最核心题型**，几乎每次考试必有（Q24/Q36/Q56/Q57）。

**出题方式**：给溶解度曲线图/表格+实验操作→判断选项正误。

**关键计算**：
1. **某温度下溶液质量** = 溶剂质量 + 溶解的溶质质量（不是加入的！）
2. **溶质质量分数** = 溶质/(溶剂+溶质) × 100%
3. **饱和溶液同温度下质量分数相等**（无论溶液多少）

**陷阱集锦**（Q56）：
- 同温度下两个饱和溶液→质量分数相等（①=②<③）
- 溶液质量≠加入的溶质+溶剂（有未溶固体）
- 升温→溶解度↑→更多固体溶解→质量分数↑

**溶解度数据表**（Q36/Q57）：
- 先根据数据计算温度对应的溶解度
- 判断是否饱和：剩余固体>0→饱和；恰好全溶且溶质=溶解度×水/100→恰好饱和
- 降温结晶：降温后按低温溶解度算析出量

### 考法2：海水制盐+粗盐精制流程

**出题方式**：给流程图→判断说法正误（Q11/Q40/Q41）。

**海水→粗盐**（物理变化）：
- 储水池→蒸发池(①蒸发浓缩)→结晶池→粗盐+母液
- 母液是NaCl饱和溶液，含MgCl₂/CaCl₂等重要化工原料
- **陷阱**：NaCl质量分数不是一直增加（饱和后不变）

**粗盐→精盐**（物理+化学）：
- 溶解→过滤(除不溶杂质)→化学除杂(除Mg²⁺/SO₄²⁻/Ca²⁺)→过滤→酸化→蒸发
- **除杂顺序必须正确**：NaOH(除Mg²⁺)→BaCl₂(除SO₄²⁻)→Na₂CO₃(除Ca²⁺+多余Ba²⁺)→过滤→HCl(除OH⁻/CO₃²⁻)
- **白色滤渣含4种**：Mg(OH)₂+BaSO₄+CaCO₃+BaCO₃（不是3种！）

### 考法3：溶液概念辨析

**出题方式**：给海水/溶液相关描述→判断正误（Q11）。

| 概念 | 正确表述 | 错误表述 |
|------|----------|----------|
| 海水 | 溶液，水作溶剂 | — |
| NaCl溶解 | 电离成Na⁺+Cl⁻ | "分解反应"✗(电离≠分解) |
| CaCl₂读法 | 氯化钙 | "二氯化钙"✗ |
| 命名规则 | 某化某(非变价) | 不读数字 |

### 考法4：配制溶液操作+误差分析

**出题方式**：给配制步骤/操作图→判断正误或误差方向（Q16/Q22/Q58）。

**基本步骤**：计算→称量→量取→溶解→装瓶贴签

**误差分析**（Q22）：
| 操作 | 对质量分数影响 |
|------|---------------|
| 称量含杂质 | ↓（溶质偏少） |
| 烧杯有水 | ↓（溶剂偏多） |
| 装瓶时溅出 | 不变（溶液均一） |
| 量水仰视 | ↓（水偏多） |

**NaOH特殊注意事项**：腐蚀性→玻璃器皿称量；易潮解→快速称量

### 考法5：物质溶于水的热量变化

**出题方式**：实验设计利用温度变化验证物质（Q44）。

- NaOH溶于水放热、NH₄NO₃溶于水吸热、NaCl基本不变
- 可利用KNO₃晶体增减（溶解度随温度变化）反向推X溶于水的热量变化
