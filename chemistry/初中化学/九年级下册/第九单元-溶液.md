# 第九单元 溶液

> 人教版（2024）九年级下册 | 中考必考（溶解度曲线+溶质质量分数计算）

## 课题1 溶液及其应用

### 溶液的定义
- **溶液**：一种或几种物质分散到另一种物质里形成的均一、稳定的混合物
- 溶质 + 溶剂 → 溶液
- 特征：**均一性、稳定性**、混合物

### 溶质与溶剂的判断
- 固体/气体 + 液体 → 固体/气体是溶质，液体是溶剂
- 两种液体互溶 → 量多的为溶剂（但只要有水，水永远是溶剂）
- 命名：溶质的溶剂溶液，如碘的酒精溶液（碘酒）

### 溶解现象
- 物质溶解时常伴随**温度变化**：
  - NaOH 溶于水 → 温度**升高**
  - NH₄NO₃ 溶于水 → 温度**降低**
  - NaCl 溶于水 → 温度变化不大

### 乳化作用
- 乳化剂使油以小液滴形式分散在水中（如洗洁精洗油污）
- 乳化 ≠ 溶解（乳浊液不均一、不稳定）

## 课题2 溶解度

### 饱和溶液与不饱和溶液
- **饱和溶液**：在一定温度下、一定量溶剂中，不能再溶解某种溶质的溶液
- 相互转化（大多数固体）：
  ```
  不饱和 ⇌ 饱和
    加溶质/降温/蒸发溶剂 →
    ← 加溶剂/升温
  ```

### 固体溶解度
- **定义**：在一定温度下，某固体在 100g 溶剂里达到饱和状态时所溶解的质量（单位：g）
- 大多数固体溶解度随温度升高而**增大**（KNO₃ 型）
- NaCl 溶解度几乎不受温度影响
- **Ca(OH)₂ 溶解度随温度升高反而降低**（特殊）

### 气体溶解度
- 随温度升高而减小，随压强增大而增大
- 喝汽水打嗝（温度↑→气体溶解度↓）、开汽水瓶冒泡（压强↓→溶解度↓）

### 溶解度曲线图读图（中考必考）
- 交点 → 该温度下两种物质溶解度相等
- 曲线上方 → 饱和溶液 + 有未溶固体
- 分离方法：
  - KNO₃（陡升型）→ **降温结晶**（冷却热饱和溶液）
  - NaCl（缓升型）→ **蒸发结晶**

## 课题3 溶质的质量分数

### 基本公式
- **ω = m(溶质) / m(溶液) × 100%**
- m(溶液) = m(溶质) + m(溶剂)
- 饱和溶液中：ω = S / (S+100) × 100%（S 为溶解度）

### 溶液稀释
- 稀释前后**溶质质量不变**
- m₁ω₁ = m₂ω₂

### 配制一定溶质质量分数的溶液
**步骤**：计算 → 称量（托盘天平称固体 / 量筒量液体）→ 溶解（烧杯 + 玻璃棒搅拌）→ 装瓶贴标签

**误差分析**：
- 称量时砝码和药品放反（用游码）→ 偏小
- 量取水时仰视读数 → 水偏多 → 偏小
- 转移时有洒落 → 偏小
