## 课题1 化学与人体健康

### 六大营养素
| 营养素 | 作用 | 来源 |
|--------|------|------|
| 蛋白质 | 构成细胞、生长发育 | 鱼、肉、蛋、奶、豆 |
| 糖类 | 供能（主要） | 米、面、土豆、甘蔗 |
| 油脂 | 备用能源 | 动植物油、肥肉 |
| 维生素 | 调节代谢（不供能） | 蔬菜、水果 |
| 无机盐 | 构成组织、维持功能 | 食盐、海带、肝脏 |
| 水 | 溶剂、运输、调节体温 | — |

### 常考元素与人体健康
| 元素 | 缺乏症状 | 过量危害 |
|------|---------|---------|
| Ca | 佝偻病、骨质疏松 | 结石 |
| Fe | 贫血 | — |
| I | 甲状腺肿大（大脖子病） | 甲亢 |
| Zn | 食欲不振、发育不良 | — |
| F | 龋齿 | 氟斑牙、氟骨症 |
| Se | 免疫力下降 | 硒中毒 |

### 对人体有害的物质
- **CO**：与血红蛋白结合（毒性）
- **甲醛**：使蛋白质变性（防腐，食品中禁用）
- **亚硝酸钠 NaNO₂**：致癌物（外观像食盐！）
- **甲醇 CH₃OH**：工业酒精，有毒（致失明甚至死亡）
- **重金属盐**（含Cu²⁺、Hg²⁺、Pb²⁺等）：使蛋白质变性

### 食品中的有机物
- 葡萄糖 C₆H₁₂O₆：供能物质
- 淀粉 (C₆H₁₀O₅)n：遇碘变蓝（检验淀粉）
- 蛋白质：遇浓硝酸变黄

## 课题2 化学与可持续发展

### 三大合成材料
| 材料 | 代表物 |
|------|--------|
| 塑料 | 聚乙烯(PE)、聚氯乙烯(PVC) |
| 合成纤维 | 涤纶、尼龙、腈纶 |
| 合成橡胶 | 丁苯橡胶 |

鉴别羊毛/棉/合成纤维：灼烧闻气味 — 羊毛（烧焦羽毛味）、棉（烧纸味）、合成纤维（特殊气味，硬结）

### 环境污染
- **酸雨**：pH<5.6 → SO₂、NO₂ → 主要来自煤燃烧和汽车尾气
- **温室效应**：CO₂、CH₄
- **臭氧层破坏**：氟利昂（CFCs）
- **水体富营养化**：含N、P废水 → 藻类疯长 → 赤潮/水华
- **白色污染**：废弃塑料 → 减少使用、回收利用、使用可降解塑料

### 绿色化学
- 原料、生产过程、产品均无毒无害
- 原子利用率 100%

## 中考考法归纳（基于真题分析）

### 考法1：NaCl的用途综合

**出题方式**：给NaCl相关说法→判断正误（Q12）。

**关键辨析**：
- 生理盐水浓度→0.9%（非高浓度）
- NaCl可制特色食盐(加碘/加钙等)✓
- 饱和NaCl溶液电解→NaOH+Cl₂+H₂(质量守恒)✓
- 成人NaCl摄入量≤5g/天✓

### 考法2：药品说明书+化学式计算

**出题方式**：给药品说明书(含结构式/分子式/分子量)→计算判断（Q42/Q49）。

**盐酸二甲双胍**（C₄H₁₁N₅·HCl）：
- 分子量=M(C₄H₁₁N₅)+M(HCl)=129+36.5=165.63
- 每片含HCl=0.5×(36.5/165.63)≈0.11g（非0.19g！）
- C:N质量比=48:70=24:35✓

**对乙酰氨基酚**（C₈H₉NO₂）：
- 含碳化合物=有机物✓
- 分子中不含NO₂分子（原子在同一个分子中）
- H:N质量比=9:14（非9:1）
- 过量致肝损伤→不可长期大量服用

### 考法3：化肥知识综合

**出题方式**：选肥料/判断化肥作用（Q51/Q60）。

**三大营养元素功能**：
- **N(氮)**：促叶生长
- **P(磷)**：促根/花/果实，增强抗寒抗旱能力
- **K(钾)**：增强抗病虫害/抗倒伏能力

**常见化肥**：
- 氮肥：CO(NH₂)₂(尿素)、NH₄NO₃
- 钾肥：K₂CO₃(草木灰)、KCl、K₂SO₄
- 磷肥：Ca₃(PO₄)₂
- 复合肥：KNO₃

> **学生误区**：尿素是氮肥但不能增强抗旱抗寒（那是磷肥的功能！）

### 考法4：化学与健康

**出题方式**：化学物质在人体/生活中的作用（Q12/Q49）。

- NaCl维持生理活动但过量→高血压风险
- 盐酸二甲双胍→2型糖尿病(肥胖型)常用药
- 对乙酰氨基酚→解热镇痛,过量致肝损伤
- 泡菜高盐→少食提醒

### 考法5：工业流程·尿素合成

**出题方式**：CO₂+NH₃→尿素微观过程→判断正误（Q60）。

**两步法**：
```
① CO₂+2NH₃ → NH₂COONH₄ (氨基甲酸铵,化合反应)
② NH₂COONH₄ → CO(NH₂)₂+H₂O (尿素,分解反应)
总：CO₂+2NH₃ → CO(NH₂)₂+H₂O
```
**陷阱**：步骤①CO₂:NH₃分子比=1:2(非1:1)；步骤②原子个数守恒！
