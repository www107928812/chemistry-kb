# 第八单元 金属和金属材料

> 人教版（2024）九年级下册 | 中考必考（金属活动性顺序）

## 课题1 金属材料

### 金属的物理性质
- 金属光泽、导电性、导热性、延展性
- 常温下大多数是固体（Hg 是液体）
- 大多数是银白色（Cu 紫红色，Au 黄色）

### 合金
- 合金是混合物（至少含一种金属）
- 合金硬度一般**大于**纯金属，熔点一般**低于**纯金属
- **生铁**：含碳量 2%~4.3%；**钢**：含碳量 0.03%~2%
- 不锈钢：含 Cr、Ni（抗腐蚀）

## 课题2 金属的化学性质

### 金属 + O₂
- 大多数金属能与 O₂ 反应（Au 不能）
- 铝在空气中形成致密氧化膜 Al₂O₃（有抗腐蚀性）

### 金属 + 酸
- 条件：金属排在 (H) 前面
- Zn + H₂SO₄ → ZnSO₄ + H₂↑
- Fe + 2HCl → FeCl₂ + H₂↑
- **注意**：铁与酸生成 **Fe²⁺（亚铁盐）**，不是 Fe³⁺！

### 金属 + 盐溶液
- 条件：前面金属置换后面金属（K、Ca、Na 除外——先与水反应）

### 金属活动性顺序（必背）
```
K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
→→→→→→→→→→→→→→→ 金属活动性越来越弱 →→→→→→→→→→→→→→→
```

**规律**：
1. (H) 前面的金属能从酸中置换出 H₂
2. 前面的金属能把后面的金属从其盐溶液中置换出来
3. K、Ca、Na 会先与水反应，不能置换盐溶液中的金属

### 湿法炼铜
- Fe + CuSO₄ → FeSO₄ + Cu
- 不能用铁桶装 CuSO₄ 溶液

## 课题3 金属资源的利用和保护

### 铁的冶炼（工业）
- 原料：铁矿石、焦炭、石灰石
- 原理：Fe₂O₃ + 3CO →高温→ 2Fe + 3CO₂
- CO 作用：还原剂、提供热量

### 金属的锈蚀与防护
- **生锈条件**：O₂ + H₂O（缺一不可）
- Cu 生铜绿：O₂ + H₂O + CO₂
- **防锈方法**：
  1. 保持干燥
  2. 涂保护层（油漆、涂油、电镀）
  3. 制成合金（如不锈钢）
